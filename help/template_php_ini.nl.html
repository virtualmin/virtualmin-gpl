Template PHP configuratie file
Standaard wanneer een virtuele server PHP scripts uitvoert als de server eigenaar, zal Virtualmin de systeem globale PHP configuratie file kopieren (normaal gesproken /etc/php.ini) naar de domein ~/etc directory. Hierdoor heeft u de mogelijkheid om PHP opties in te stellen op een per-server basis.

Deze instelling kan worden gebruikt om een ander pad in te vullen om de basis PHP configuratie file te kopieren. Als toevoeging kunt u Virtualmin substituties in deze template file gebruiken zoals ${DOM}, die dan word vervangen zodra hij word gekopieerd. Andere bruikbare substituties die beschikbaar zijn :

${DOM} De domein naam, zoals foo.com
${USER} De server administratie login, zoals foo
${HOME} De server home directory, zoals /home/foo
${EMAILTO} De virtuele server contact email adres
Als toevoeging kunnen ook blokken zoals ${IF-MAIL}...${ENDIF-MAIL} and ${IF-WEB}...${ELSE-WEB}...{$ENDIF-WEB} worden gebruikt.

Databázový server MySQL (pre databázu)
Ak je povolené, účet vlastníka domény bude mať prístup k modulu MySQL Webmin. Poskytuje priateľské GUI k databázam, ktoré patria vlastníkovi domény. Pomocou tohto modulu a príslušných privilégií môže vlastník domény vytvárať nové databázy, upravovať existujúce databázy alebo vykonávať dotazy na údaje v databáze.

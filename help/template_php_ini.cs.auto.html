Šablona konfiguračního souboru PHP
Ve výchozím nastavení, když virtuální server spustí skripty PHP jako vlastník serveru, Virtualmin zkopíruje globální konfigurační soubor PHP systému (obvykle /etc/php.ini ) do adresáře ~/etc v doméně. To umožňuje, aby byly možnosti PHP nastaveny odlišně pro jednotlivé servery.

Toto nastavení lze použít k zadání jiné cesty ke kopírování základního konfiguračního souboru PHP. Kromě toho můžete v tomto souboru šablony použít substituce Virtualmin jako ${DOM}, které budou při kopírování nahrazeny. Mezi další dostupné substituce patří:

${DOM} Název domény, například foo.com
${USER} Přihlašovací údaje pro správu serveru, například foo
${HOME} Domovský adresář serveru, například /home/foo
${EMAILTO} Kontaktní e-mailová adresa virtuálního serveru
Kromě toho lze použít podmíněné bloky jako ${IF-MAIL} ... ${ENDIF-MAIL} a ${IF-WEB} ... ${ELSE-WEB} ... {$ENDIF-WEB} .

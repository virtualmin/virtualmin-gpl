Hébergement pour les enregistrements DNS
Si vous avez configuré d'autres fournisseurs Cloud DNS, cette option vous permet de déplacer l'hébergement DNS de ce domaine entre eux, ou de l'hébergement local vers le cloud. Notez que cela peut nécessiter la mise à jour des adresses IP de ce serveur de noms avec votre registraire DNS après avoir effectué le déplacement.

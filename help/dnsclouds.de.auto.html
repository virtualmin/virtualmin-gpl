Cloud-DNS-Anbieter
Auf dieser Seite können Sie Virtualmin so konfigurieren, dass andere DNS-Dienste als Ihr eigenes System verwendet werden. Diese bieten normalerweise eine höhere Verfügbarkeit und schnellere DNS-Suche für Clients zu einem Preis pro gehosteter Domain. Sobald hier ein Anbieter eingerichtet wurde, können Sie ihn auf der Seite Servervorlagen als Standard für neue virtuelle Server festlegen.

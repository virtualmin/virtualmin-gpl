Créer des enregistrements DNS TLSA pour les certificats SSL?
Lorsqu'il est activé, Virtualmin ajoute un enregistrement DNS TLSA pour tous les noms d'hôte qui utilisent des certificats SSL pour les serveurs virtuels nouvellement créés. Cela permet aux clients de valider que le certificat SSL présenté sur votre serveur Web est le même que celui hébergé sur votre système, ce qui empêche certaines attaques man-in-the-middle.

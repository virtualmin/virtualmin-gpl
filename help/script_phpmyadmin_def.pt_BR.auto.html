Usar phpMyAdmin como padrão do sistema
Quando ativada, esta instalação do phpMyAdmin é definida como padrão em todo o sistema. Um botão "Gerenciar no phpMyAdmin" será exibido na página "Editar Bancos de Dados" para domínios que não possuem sua própria instalação do phpMyAdmin.

A versão do phpMyAdmin utilizada depende do que está instalado no sistema:

  • Se o subservidor tiver sua própria instalação do phpMyAdmin, essa instalação será utilizada.
  • Caso contrário, se o servidor virtual de nível superior tiver o phpMyAdmin instalado, essa instalação será usada tanto para o servidor de nível superior quanto para quaisquer subservidores que não possuam sua própria instalação.
  • Caso contrário, será utilizada a instalação padrão do phpMyAdmin do sistema.

Isso pode ser útil quando você deseja uma instância compartilhada do phpMyAdmin (por exemplo, em um nome de host dedicado como pma.example.tld ), permitindo ainda instalações por servidor virtual quando necessário.

Atenção: utilize uma instância padrão do phpMyAdmin compartilhada em um hostname dedicado somente se você confiar plenamente no servidor virtual que a hospeda. Quando outros servidores virtuais não possuem seu próprio phpMyAdmin instalado, eles utilizarão essa instância padrão, o que pode significar que seus dados de login do MySQL serão repassados para ela. Se o servidor virtual que hospeda o phpMyAdmin estiver comprometido, mal configurado ou controlado por alguém em quem você não confia totalmente, essas credenciais poderão ser expostas.

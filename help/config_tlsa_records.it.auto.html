Creare record DNS TLSA per certificati SSL?
Quando abilitato, Virtualmin aggiungerà un record DNS TLSA per tutti i nomi host che utilizzano certificati SSL per i server virtuali appena creati. Ciò consente ai client di verificare che il certificato SSL presentato sul tuo server web sia lo stesso di quello ospitato sul tuo sistema, il che previene alcuni attacchi man-in-the-middle.

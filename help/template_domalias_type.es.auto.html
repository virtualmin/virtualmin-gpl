Prefijo para dominio de alias
Cuando se crea un dominio de alias automático para un nuevo servidor virtual, esta opción determina si la primera parte del dominio de alias es el nombre de dominio completo del servidor virtual, o solo la parte anterior al primer punto. Por ejemplo, si el dominio para crear bajo es isp.com y el nuevo dominio es foo.com , al seleccionar el nombre de dominio completo , el alias será foo.com.isp.com , mientras que al seleccionar la primera parte del dominio , ser foo.isp.com .

Alternativamente, puede seleccionar la plantilla Virtualmin e ingresar una expresión arbitraria que utilice variables de plantilla como ${DOM}, ${USER} o ${PREFIX}. Se pueden crear prefijos generados aleatoriamente de hasta 9 dígitos con una variable como ${RANDOM5}, que crea un número aleatorio de 5 dígitos.

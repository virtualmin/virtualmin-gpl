Toegestane login type
Deze instelling bepaald hoe de administrator voor dit domein moet inloggen door het veranderen van zijn Unix shell. De standaard beschikbare opties zijn :

Alleen email
Zowel FTP als SSH inloggen is geblokkeerd, maar POP3 en IMAP zal werken.
Email en FTP
De gebruiker kan verbinden via FTP, maar niet via SSH of telnet.
Email, FTP en SSH
Inloggen via FTP, SSH en telnet is toegestaan.
Deze kunnen echter ook worden bewerkt bij de hoofd administrator op de Eigen Shells pagina.

Omdat deze functie werkt door het wijzigen van de virtuele server Unix gebruiker's shell, gaat het er vanuit dat uw FTP server de /etc/shells file gebruikt voor authenticatie.

Nome do host para registro MX
Por padrão, o registro MX do DNS (que controla onde o email é entregue) para novos domínios será mail. $ DOMAIN, ao qual será atribuído o mesmo endereço IP do site do servidor virtual. No entanto, você pode usar este campo para especificar um nome de host diferente para o servidor de email - normalmente é usado para ter um registro MX consistente em todos os domínios. Certifique-se de inserir um nome de host que seja resolvido no seu sistema Virtualmin, caso contrário, o e-mail não será entregue.

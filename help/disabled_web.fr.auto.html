Site Web pour handicapés HTML
Ce champ peut être utilisé pour personnaliser le message qui apparaît lors de la connexion à un site Web pour un serveur virtuel désactivé. Le message par défaut indique simplement Site Web désactivé . Vous pouvez utiliser des substitutions typiques comme ${DOM} et ${USER} pour obtenir des détails sur le domaine désactivé, ${DISABLED_WHY} pour obtenir la raison pour laquelle le domaine a été désactivé et ${DISABLED_TIME} pour obtenir l'heure à laquelle le désactiver a été fait.

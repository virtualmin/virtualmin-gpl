Provider di Cloud DNS
Questa pagina consente di configurare Virtualmin per utilizzare servizi DNS diversi dal proprio sistema. Questi in genere forniscono una maggiore disponibilità e ricerche DNS più veloci ai client, a un costo per dominio ospitato. Una volta che un provider è stato impostato qui, puoi impostarlo come predefinito per i nuovi server virtuali nella pagina dei modelli di server.

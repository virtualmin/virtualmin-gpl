File modello per e-mail a caselle di posta aggiornate
Se le cassette postali devono ricevere automaticamente un messaggio quando vengono aggiornate, è possibile specificarlo qui. Può essere l'impostazione predefinita, che può essere modificata utilizzando la pagina E-mail della casella postale aggiornata di Virtualmin oppure è possibile fornire un altro file di testo in qualsiasi punto del sistema. Sostituzione delle variabili usando le seguenti variabili: ${MAILBOX}, ${USER}, ${DOM}, ${FTP} e ${HOME}. È inoltre possibile utilizzare gli operatori del blocco logico speciale $IF-MAIL ... $ENDIF-MAIL e $IF-WEB ... $ELSE-WEB ... {$ENDIF-WEB} per consentire al messaggio di creare un messaggio appropriato in base alle funzionalità a cui l'utente ha accesso.

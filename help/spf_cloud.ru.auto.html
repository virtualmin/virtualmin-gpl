Хостинг для DNS-записей
Если у вас настроены другие поставщики облачных DNS, этот параметр позволяет перемещать DNS-хостинг для этого домена между ними или с локального хостинга в облако. Обратите внимание, что для этого может потребоваться обновить IP-адреса этого сервера имен у вашего регистратора DNS после выполнения перемещения.

Modello server
Un modello di server definisce le impostazioni predefinite per i nuovi domini o per le funzionalità aggiunte ai domini esistenti. Quasi tutti i campi di testo e le caselle sottostanti consentono di utilizzare le variabili che verranno sostituite in base alle impostazioni del server virtuale al momento dell'utilizzo del modello. Alcune delle variabili più utili sono:

${DOM} Il nome di dominio, ad esempio foo.com
${PREFIX} La prima parte del nome di dominio, come foo
${USER} L'utente che possiede il dominio, ad esempio foo
${IP} L'indirizzo IP assegnato al server virtuale, ad esempio 192.168.1.1
${HOME} La home directory dell'utente di dominio, come /home/foo
${UID} L'ID utente di sistema dell'utente del proprietario del dominio, ad esempio 502 .
Inoltre, possono essere usati blocchi condizionali come $IF-MAIL ... $ENDIF-MAIL e $IF-WEB ... $ELSE-WEB ... {$ENDIF-WEB} .

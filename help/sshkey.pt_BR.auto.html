Chave pública SSH
Este campo pode ser usado para conceder acesso SSH autenticado por meio de uma chave pública para o usuário Unix do novo servidor virtual. As opções são:

Sem chave padrão
Não configure uma chave SSH. Somente logins com senha serão permitidos.

Gerar chaves públicas e privadas
Crie um novo par de chaves SSH para este servidor virtual e permita logins usando essa chave. O proprietário do servidor deve copiar a chave privada após a criação.

Use a chave pública abaixo
Os logins SSH serão permitidos usando a chave privada que corresponde à chave pública inserida na caixa de texto abaixo.

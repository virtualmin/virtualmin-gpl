Cuota de CPU
Establece un límite estricto para el tiempo total de CPU para este usuario, como porcentaje de un núcleo de CPU. 100% significa que el usuario puede usar hasta el valor completo de un núcleo de tiempo de CPU. 200% significa hasta el valor de dos núcleos. 25% significa un cuarto del valor de un núcleo, compartido entre todos los procesos del usuario.

Nakonfigurujte Apache pro použití
Tato možnost určuje, zda má být Apache nakonfigurován tak, aby používal jeden soubor, který kombinuje certifikát SSL pro virtuální server a certifikát CA, nebo zda mají být rozděleny do samostatných souborů. Výchozí nastavení je kombinovat je pouze při spuštění Apache 2.4.8 nebo novějšího.

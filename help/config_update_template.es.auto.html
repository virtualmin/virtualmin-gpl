Archivo de plantilla para correo electrónico a buzones actualizados
Si los buzones se deben enviar automáticamente un mensaje cuando se actualizan, puede especificarlo aquí. Puede ser el predeterminado, que puede editarse utilizando la página Correo electrónico de buzón actualizado de Virtualmin, o puede proporcionar otro archivo de texto en cualquier parte del sistema. Sustitución de variables usando las siguientes variables: ${MAILBOX}, ${USER}, ${DOM}, ${FTP} y ${HOME}. También puede usar los operadores de bloques lógicos especiales $IF-MAIL ... $ENDIF-MAIL y $IF-WEB ... $ELSE-WEB ... {$ENDIF-WEB} para permitir que su mensaje cree un mensaje apropiado según las características a las que el usuario tiene acceso.

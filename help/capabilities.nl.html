Mag de virtuele server bewerken
Hiermee kunnen virtuele server opties bewerkt worden, het wachtwoord en de omschrijving.
Mag gebruikers beheren
Hiermee kunnen mailbox, FTP en database gebruikers worden bekeken, bewerkt, aangemaakt en verwijdert.
Mag aliassen beheren
Hiermee kunnen email aliassen worden bekeken, bewerkt, aangemaakt en verwijdert.
Mag databases beheren
Hiermee kunnen extra MySQL en PostgreSQL databases worden toegevoegd aan deze virtuele server of worden verwijdert.
Mag scripts installeren
Hiermee kunnen 3e-partij scripts worden geinstalleerd in deze virtuele server.
Mag IP adressen wijzigen
Indien deze server prive IP adres heeft kan het met deze optie worden gewijzigd.
Mag SSL certificaten beheren
Hiermee kan een SSL certificaat en sleutel voor deze server worden ge-upload of opnieuw gegenereerd.
Mag doorstuur frame bewerken
Voor alleen-doorsturen websites, kan met deze optie de bestemming en HTML voor het doorstuur frame worden bewerkt.
Mag extra admins beheren
Staat toe dat extra server eigenaar administrator accounts kunnen worden bekeken, aangemaakt, bewerkt en verwijdert.
Mag spam en virus aflevering configureren
Hiermee kunnen spam en virus aflever bestemmingen en de score worden bewerkt.
Mag PHP versies selecteren
Voor domeinen waar PHP werkt via CGI of fastCGI kan er dan een per-directory PHP versie worden ingesteld.
Mag email instellingen bewerken
Hiermee kan de configuratie worden aangepast van automatisch BCC'en, alias kopie mode en andere email-gerelateerde opties.
Mag backup's maken
Toestaan dat een eigen virtuele server backup word verstuurd naar een remote FTP of SSH server.
Mag backup's plannen
Toestaan dat een virtuele server backup word gemaakt in een reguliere planning.
Mag backup's herstellen
Toestaan dat een home directory en database inhoud van een virtuele server die in bezit is worden hersteld.
Mag gedeelde IP adressen selecteren
Toestaan dat een globaal gedefinieerde gedeelde IP adressen worden geselecteerd wanneer er een virtuele server word aangemaakt.
Mag een 'catchall' alias aanmaken
Aanmaken toestaan van een email alias die alle email doorstuurt naar een domein, naar een ander domein of een adres.
Mag de web pagina bewerker gebruiken
Hierdoor kunnen web pagina's worden bewerkt via de Virtualmin UI.
Mag database bewerken voor remote hosts
Toestaan dat remote client systemen kunnen verbinden met de MySQL databases van de domeinen.
Mag virtuele servers uitschakelen
Toestaan dat servers tijdelijk worden uitgeschakeld.
Mag virtuele servers verwijderen
Toestaan dat eigen servers compleet worden verwijdert, inclusief alle sub-servers en aliassen.

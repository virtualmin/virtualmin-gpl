Nome do banco de dados padrão
O Virtualmin permite criação e permissões de banco de dados muito flexíveis. Este campo permite especificar o nome padrão dos bancos de dados criados como parte dos domínios criados a partir deste modelo. Por padrão, o Virtualmin usa a variável ${PREFIX}, que é a primeira parte do nome do domínio, e geralmente também será o nome de login do proprietário do domínio para o Webmin. Pode ser útil alterar isso, em algumas circunstâncias, para incluir um prefixo que indica o modelo do qual foi gerado ou para incluir informações adicionais sobre o banco de dados. É possível combinar variáveis e quaisquer caracteres permitidos em um banco de dados PostgreSQL e / ou MySQL (dependendo de quais bancos de dados estão disponíveis).

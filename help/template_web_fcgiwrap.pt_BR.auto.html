Execute scripts CGI usando
Quando o servidor fcgiwrap é instalado, esta opção permite que você selecione se ele será sempre usado para executar scripts CGI para novos domínios. Caso contrário, o suexec será usado se estiver instalado e configurado corretamente.

FCGIwrap geralmente é mais rápido para executar scripts CGI, mas requer que cada servidor virtual execute seu próprio processo de servidor, o que consome RAM adicional.

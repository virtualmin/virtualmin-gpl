Comando da eseguire prima di apportare modifiche a un dominio
Se si desidera eseguire alcune azioni prima di modificare un dominio (creazione, alterazione o eliminazione) è possibile specificare un comando da eseguire qui. Se devono essere eseguite più azioni, il comando da eseguire può essere uno script. Il comando ha accesso a speciali variabili di ambiente specifiche del dominio, come VIRTUALSERVER_DOM, VIRTUALSERVER_IP, VIRTUALSERVER_USER, VIRTUALSERVER_HOME, VIRTUALSERVER_PASS e altro. Qualsiasi variabile trovata nel file / etc / webmin / virtual-server / domains / $DOM sarà disponibile come variabile d'ambiente.

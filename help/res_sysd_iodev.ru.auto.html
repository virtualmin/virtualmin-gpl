устройство ввода/вывода
Выбирает блочное устройство, к которому будут применяться ограничения дискового ввода-вывода. Устройство по умолчанию определяется из файловой системы, где находится домашний каталог пользователя. Используйте пользовательское устройство, если устройство по умолчанию не подходит для вашей схемы хранения данных.

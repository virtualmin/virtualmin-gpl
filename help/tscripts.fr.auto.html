Installateurs de scripts de modèles
Cette page vous permet de spécifier des scripts tiers à installer lorsqu'un nouveau serveur virtuel est créé à l'aide de ce modèle. Pour chaque script, les informations suivantes doivent être saisies:

Scénario
Le nom et la version d'un script à installer
Chemin URL
Un chemin sous le site Web du serveur virtuel pour installer le script, tel que /phpmyadmin . Cela déterminera également le répertoire sous ~/public_html qui contient les fichiers du script.
Nom de la base de données
Nom de la base de données que le script utilisera pour stocker les données. Les options sont les suivantes:
Aucun
Cela doit être sélectionné pour les scripts qui n'ont pas besoin d'une base de données, comme Gallery .
Base de données principale
La base de données du script sera la base de données principale du serveur virtuel du type choisi.
Autre
Si vous souhaitez que ce script utilise sa propre base de données privée, cette option peut être sélectionnée et le modèle d'une base de données saisi dans le champ de texte adjacent. Par exemple, vous pouvez entrer ${DB}_phpnuke . Si la base de données résultante n'existe pas, elle sera créée dans le cadre du processus d'installation du script.
Type de DB
Le type de base de données utilisé pour les données de scripts, le cas échéant, est sélectionné.

Sous-répertoire personnel
Les nouveaux domaines (qui n'appartiennent pas aux propriétaires de domaines existants) résideront dans un sous-répertoire du répertoire personnel spécifié ci-dessus. Par défaut, le nom du répertoire correspondra au nom d'utilisateur et résidera directement dans le répertoire de base. Vous pouvez également spécifier un nom de répertoire spécial, si par exemple, vous souhaitez conserver les utilisateurs système standard séparés des utilisateurs de compte de domaine. Dans un tel cas, vous pouvez placer les domaines Virtualmin dans "/ home / domaines / $USER".

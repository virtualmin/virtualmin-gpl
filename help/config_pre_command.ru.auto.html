Команда для запуска перед внесением изменений в домен
Если вы хотите выполнить какое-либо действие перед изменением домена (создание, изменение или удаление), вы можете указать команду для запуска здесь. Если нужно выполнить несколько действий, команда для запуска может быть скриптом. Команда имеет доступ к специальным переменным среды, специфичным для домена, таким как VIRTUALSERVER_DOM, VIRTUALSERVER_IP, VIRTUALSERVER_USER, VIRTUALSERVER_HOME, VIRTUALSERVER_PASS и другие. Любая переменная, найденная в файле / etc / webmin / virtual-server / domains / $DOM, будет доступна в качестве переменной среды.

Comando para ejecutar antes de realizar cambios en un dominio
Si desea realizar alguna acción antes de modificar un dominio (creación, alteración o eliminación), puede especificar un comando para ejecutar aquí. Si se deben realizar varias acciones, el comando a ejecutar puede ser un script. El comando tiene acceso a variables de entorno especiales específicas del dominio, como VIRTUALSERVER_DOM, VIRTUALSERVER_IP, VIRTUALSERVER_USER, VIRTUALSERVER_HOME, VIRTUALSERVER_PASS y más. Cualquier variable encontrada en el archivo / etc / webmin / virtual-server / domains / $DOM estará disponible como una variable de entorno.

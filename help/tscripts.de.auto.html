Template Script Installers
Auf dieser Seite können Sie Skripts von Drittanbietern angeben, die installiert werden sollen, wenn mit dieser Vorlage ein neuer virtueller Server erstellt wird. Für jedes Skript müssen die folgenden Informationen eingegeben werden:

Skript
Der Name und die Version eines zu installierenden Skripts
URL-Pfad
Ein Pfad unter der Website des virtuellen Servers zum Installieren des Skripts, z. B. /phpmyadmin . Dadurch wird auch das Verzeichnis unter ~/public_html , das die Dateien des Skripts enthält.
DB-Name
Der Name der Datenbank, in der das Skript Daten speichert. Die Optionen sind:
Keiner
Dies sollte für Skripte ausgewählt werden, die keine Datenbank benötigen, z. B. Galerie .
Hauptdatenbank
Die Datenbank des Skripts ist die Primärdatenbank des virtuellen Servers des ausgewählten Typs.
Andere
Wenn dieses Skript eine eigene private Datenbank verwenden soll, kann diese Option ausgewählt und das Muster für eine Datenbank in das nebenstehende Textfeld eingegeben werden. Beispielsweise können Sie $ {DB} _phpnuke eingeben . Wenn die resultierende Datenbank nicht vorhanden ist, wird sie im Rahmen der Skriptinstallation erstellt.
DB-Typ
Die Art der Datenbankverwendung für die Skriptdaten, falls vorhanden.

Провайдеры облачных DNS
Эта страница позволяет вам настроить Virtualmin для использования служб DNS, отличных от вашей собственной системы. Обычно они обеспечивают более высокую доступность и более быстрый поиск DNS для клиентов по цене, приходящейся на один размещенный домен. После того, как провайдер будет настроен здесь, вы можете сделать его по умолчанию для новых виртуальных серверов на странице «Шаблоны серверов».

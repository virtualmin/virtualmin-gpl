TLSA-DNS-Einträge für SSL-Zertifikate erstellen?
Wenn diese Option aktiviert ist, fügt Virtualmin einen TLSA-DNS-Eintrag für alle Hostnamen hinzu, die SSL-Zertifikate für neu erstellte virtuelle Server verwenden. Auf diese Weise können Clients überprüfen, ob das auf Ihrem Webserver präsentierte SSL-Zertifikat mit dem auf Ihrem System gehosteten Zertifikat identisch ist, wodurch bestimmte Man-in-the-Middle-Angriffe verhindert werden.

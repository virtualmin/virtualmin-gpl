Установщики шаблонных скриптов
На этой странице вы можете указать сторонние скрипты, которые будут устанавливаться при создании нового виртуального сервера с использованием этого шаблона. Для каждого сценария необходимо ввести следующую информацию:

скрипт
Имя и версия скрипта для установки
URL-путь
Путь под веб-сайтом виртуального сервера для установки скрипта, например /phpmyadmin . Это также определит каталог в ~/public_html который содержит файлы скрипта.
Имя БД
Имя базы данных, которую скрипт будет использовать для хранения данных. Варианты:
Никто
Это следует выбирать для сценариев, которым не нужна база данных, например, Галерея .
Основная база данных
База данных сценария будет основной базой данных виртуального сервера выбранного типа.
Другой
Если вы хотите, чтобы этот сценарий использовал свою собственную частную базу данных, этот параметр можно выбрать, и шаблон для базы данных будет введен в смежное текстовое поле. Например, вы можете ввести ${DB}_phpnuke . Если полученная база данных не существует, она будет создана как часть процесса установки скрипта.
Тип БД
Тип базы данных, используемой для данных сценариев, если они выбраны.

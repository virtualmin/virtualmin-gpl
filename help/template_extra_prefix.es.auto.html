Prefijo para administradores adicionales
Este campo se puede usar para forzar que las cuentas de administración adicionales tengan nombres de usuario que comiencen con algún prefijo. Esto es útil para identificar con qué servidor está asociado un usuario particular de Webmin. Un valor común para este campo es ${USER} _, que obliga a los administradores adicionales a comenzar con el mismo nombre que el inicio de sesión de administrador principal del servidor virtual.

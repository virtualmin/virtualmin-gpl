Von Backups auszuschließende Verzeichnisse
Aus dieser Vorlage erstellte virtuelle Server schließen standardmäßig alle hier eingegebenen Verzeichnisse von Backups aus. Dies kann nützlich sein, wenn Sie wissen, dass alle Ihre Domains ähnliche Apps ausführen oder ähnliche Inhalte haben. Ausgeschlossene Verzeichnisse können nach der Domänenerstellung noch bearbeitet werden.

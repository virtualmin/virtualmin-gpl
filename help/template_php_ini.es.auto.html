Archivo de configuración PHP de plantilla
De manera predeterminada, cuando un servidor virtual ejecuta scripts PHP como propietario del servidor, Virtualmin copiará el archivo de configuración PHP global del sistema (generalmente /etc/php.ini ) en el directorio ~/etc del dominio. Esto permite que las opciones de PHP se configuren de manera diferente por servidor.

Esta configuración se puede utilizar para ingresar una ruta diferente para el archivo de configuración de PHP base para copiar. Además, puede usar sustituciones de Virtualmin como ${DOM} en este archivo de plantilla, que se reemplazará cuando se copie. Otras sustituciones útiles disponibles incluyen:

${DOM} El nombre de dominio, como foo.com
${USER} El inicio de sesión de administración del servidor, como foo
${HOME} El directorio de inicio del servidor, como /home/foo
${EMAILTO} La dirección de correo electrónico de contacto del servidor virtual
Además, se pueden usar bloques condicionales como ${IF-MAIL} ... ${ENDIF-MAIL} y ${IF-WEB} ... ${ELSE-WEB} ... {$ENDIF-WEB} .

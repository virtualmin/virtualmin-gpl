Prefisso per amministratori extra
Questo campo può essere utilizzato per imporre agli account di amministrazione extra di avere nomi utente che iniziano con un prefisso. Ciò è utile per identificare a quale server è associato un determinato utente Webmin. Un valore comune per questo campo è ${USER} _, che impone agli amministratori aggiuntivi di iniziare con lo stesso nome dell'account di accesso amministrativo principale del server virtuale.

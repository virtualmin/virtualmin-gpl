Nome host per record MX
Per impostazione predefinita, il record MX DNS (che controlla dove viene recapitata la posta elettronica) per i nuovi domini sarà mail. $DOMAIN, a cui verrà assegnato lo stesso indirizzo IP del sito Web del server virtuale. Tuttavia, è possibile utilizzare questo campo per specificare un nome host del server di posta diverso, in genere utilizzato per avere un record MX coerente in tutti i domini. Assicurati di inserire un nome host che si risolva nel tuo sistema Virtualmin, altrimenti l'e-mail non verrà recapitata.

Modelo para diretório de script padrão
Por padrão, o Virtualmin instalará novos scripts em um subdiretório com o mesmo nome que o código do script, como phpadmin . Este campo permite inserir um modelo para um diretório padrão diferente, como scripts/${SCRIPTNAME} . Todas as variáveis usuais do modelo, como $ {DOM} e $ {USER}, também podem ser usadas.

périphérique d'E/S
Permet de sélectionner le périphérique de stockage auquel appliquer les limites d'E/S disque. Le périphérique par défaut est détecté à partir du système de fichiers où se trouve le répertoire personnel de l'utilisateur. Utilisez un périphérique personnalisé si le périphérique par défaut ne convient pas à votre configuration de stockage.

電子郵件到新郵箱的模板文件
如果應該自動向創建的新郵箱發送消息，則可以在此處指定。它可以是默認值，可以使用Virtualmin的“ 新建郵箱電子郵件”頁面進行編輯，或者可以在系統上的任何位置提供另一個文本文件。使用以下變量進行變量替換：${MAILBOX}，${USER}，${DOM}，${FTP}和${HOME}。您還可以使用特殊的邏輯塊運算符$IF-MAIL ... $ENDIF-MAIL和$IF-WEB ... $ELSE-WEB ... {$ENDIF-WEB}允許您的消息創建適當的消息根據用戶有權使用的功能。如果您使用的是DAV，則可以使用${IF-VIRTUALMIN-DAV} .. ${ENDIF-VIRTUALMIN-DAV}塊來顯示不同的文本，具體取決於用戶是否激活了DAV。

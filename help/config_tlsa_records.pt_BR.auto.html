Criar registros DNS TLSA para certificados SSL?
Quando ativado, o Virtualmin adicionará um registro DNS TLSA para todos os nomes de host que usam certificados SSL para servidores virtuais recém-criados. Isso permite que os clientes validem se o certificado SSL apresentado em seu servidor da web é o mesmo que aquele hospedado em seu sistema, o que evita certos ataques man-in-the-middle.

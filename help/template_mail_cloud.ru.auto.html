Поставщик облачной доставки почты по умолчанию
Этот параметр определяет, будет ли электронная почта для новых доменов доставляться напрямую в целевые системы через SMTP или через выбранного облачного почтового провайдера. Выбранный провайдер должен быть уже правильно настроен, чтобы его можно было использовать.

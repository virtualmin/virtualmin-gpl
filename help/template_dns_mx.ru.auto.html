Имя хоста для записи MX
По умолчанию запись DNS MX (которая управляет доставкой электронной почты) для новых доменов будет представлять собой почту. $DOMAIN, которой будет присвоен тот же IP-адрес, что и веб-сайту виртуального сервера. Однако вы можете использовать это поле, чтобы указать другое имя хоста почтового сервера - обычно оно используется для создания согласованной записи MX во всех доменах. Обязательно введите имя хоста, которое разрешается в вашей системе Virtualmin, иначе электронное письмо не будет доставлено.

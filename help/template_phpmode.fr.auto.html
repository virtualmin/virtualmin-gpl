Mode d'execution PHP par défaut
Cette option détermine si les scripts PHP dans les nouveaux serveurs virtuels sont exécutés en tant qu'utilisateur du serveur Web Apache (généralement httpd ou apache ), ou en tant qu'utilisateur Unix propriétaire du serveur virtuel. Le premier est plus rapide car il signifie qu'aucun processus supplémentaire ne doit être démarré pour exécuter le code PHP, mais moins sécurisé lorsque les scripts lisent et écrivent des fichiers en tant qu'utilisateur autre que le propriétaire du domaine.

Si votre système le prend en charge, l'option FCGId est la meilleure, car elle exécute rapidement les scripts PHP et en tant que propriétaire du domaine. Sinon, à moins que votre système ne soit lourdement chargé ou que la sécurité entre les utilisateurs ne soit pas un problème, l'option wrapper CGI est recommandée.

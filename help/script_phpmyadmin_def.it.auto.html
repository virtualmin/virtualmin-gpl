Utilizzare come predefinito del sistema phpMyAdmin
Se abilitata, questa installazione di phpMyAdmin viene impostata come predefinita a livello di sistema. Un pulsante "Gestisci in phpMyAdmin" verrà visualizzato nella pagina "Modifica database" per i domini che non dispongono di una propria installazione di phpMyAdmin.

Il tipo di phpMyAdmin utilizzato dipende da cosa è installato:

  • Se il sottoserver ha la propria installazione di phpMyAdmin, viene utilizzata tale installazione
  • In caso contrario, se il server virtuale di livello superiore ha phpMyAdmin installato, tale installazione viene utilizzata sia per il server di livello superiore che per tutti i sottoserver che non hanno un proprio
  • Altrimenti verrà utilizzata l'installazione predefinita di phpMyAdmin del sistema

Ciò può essere utile quando si desidera un'istanza condivisa di phpMyAdmin (ad esempio, sotto un nome host dedicato come pma.example.tld ) consentendo comunque installazioni per server virtuale quando necessario.

Attenzione, utilizzate un phpMyAdmin condiviso "predefinito di sistema" con un nome host dedicato solo se vi fidate completamente del server virtuale che lo ospita. Quando altri server virtuali non hanno un proprio phpMyAdmin installato, utilizzeranno questa istanza predefinita, il che può significare che le loro credenziali di accesso MySQL verranno passate a quest'ultimo per l'accesso. Se il server virtuale di hosting è compromesso, configurato in modo errato o controllato da qualcuno di cui non vi fidate completamente, tali credenziali potrebbero essere esposte.

Préfixe pour les administrateurs supplémentaires
Ce champ peut être utilisé pour forcer les comptes d'administration supplémentaires à avoir des noms d'utilisateur commençant par un préfixe. Ceci est utile pour identifier le serveur auquel un utilisateur Webmin particulier est associé. Une valeur courante pour ce champ est ${USER} _, ce qui oblige les administrateurs supplémentaires à démarrer avec le même nom que la connexion administrateur principale du serveur virtuel.

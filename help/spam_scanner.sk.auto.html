Program na skenovanie vírusov
Možné možnosti pre toto pole sú:
Samostatný skener ( clamscan )
Toto je zvyčajne predvolené nastavenie a povie Virtualminu, aby použil skenovací program ClamAV, ktorý vykonáva všetku prácu sám. Toto bude určite fungovať, ale môže to byť pomalšie ako ďalšia možnosť.
Serverový skener ( clamdscan )
Túto možnosť je možné vybrať, ak je na vašom systéme spustený server na skenovanie vírusov clamd. Vo všeobecnosti je lepší, pretože vírusová databáza ClamAV je pomerne veľká a server ju potrebuje načítať do pamäte iba raz.
Skener vzdialeného servera ( clamd-stream-client )
Túto možnosť je možné vybrať, ak server na skenovanie vírusov clamd beží na vzdialenom systéme a bol nakonfigurovaný tak, aby akceptoval sieťové pripojenia od iných hostiteľov. Je to užitočné na presunutie úlohy skenovania na iný systém, ako je ten, ktorý prijíma e-maily.
Iný program
Táto možnosť vám umožňuje povedať Virtualminu, aby používal nejaký úplne iný program, ako napríklad váš vlastný obalový skript pre ClamAV.

Fichier modèle pour le courrier électronique vers les nouvelles boîtes aux lettres
Si de nouvelles boîtes aux lettres créées doivent recevoir automatiquement un message, vous pouvez le spécifier ici. Il peut s'agir de la valeur par défaut, qui peut être modifiée à l'aide de la page New Mailbox Email de Virtualmin, ou vous pouvez fournir un autre fichier texte n'importe où sur le système. Substitution de variables à l'aide des variables suivantes: ${MAILBOX}, ${USER}, ${DOM}, ${FTP} et ${HOME}. Vous pouvez également utiliser les opérateurs de bloc logique spéciaux $IF-MAIL ... $ENDIF-MAIL et $IF-WEB ... $ELSE-WEB ... {$ENDIF-WEB} pour permettre à votre message de créer un message approprié en fonction des fonctionnalités auxquelles l'utilisateur a accès. Si vous utilisez DAV, le bloc ${IF-VIRTUALMIN-DAV} .. ${ENDIF-VIRTUALMIN-DAV} peut être utilisé pour faire apparaître un texte différent selon que l'utilisateur a ou non DAV activé.

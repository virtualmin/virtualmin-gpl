Provedores de Cloud DNS
Esta página permite que você configure o Virtualmin para usar outros serviços DNS além do seu próprio sistema. Normalmente, eles fornecem maior disponibilidade e pesquisas de DNS mais rápidas para os clientes, por um custo por domínio hospedado. Depois que um provedor for configurado aqui, você pode torná-lo o padrão para novos servidores virtuais na página Modelos de servidor.

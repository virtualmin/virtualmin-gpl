Префикс для дополнительных администраторов
Это поле можно использовать, чтобы заставить дополнительные учетные записи администраторов иметь имена пользователей, начинающиеся с некоторого префикса. Это полезно для определения того, с каким сервером связан конкретный пользователь Webmin. Общее значение для этого поля - ${USER} _, что заставляет дополнительных администраторов начинаться с того же имени, что и основной логин администратора виртуального сервера.

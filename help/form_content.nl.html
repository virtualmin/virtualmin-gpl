Eerste web pagina aanmaken?
Om Virtualmin een set met basis web pagina's te laten maken voor deze virtuele server, selecteert u Ja, met de tekst hieronder optie en vult u de HTML tekst in het veld hieronder die moet verschijnen op de index pagina. Dit zal gecombineerd worden met de stijl die geselecteerd is in Stijl voor eerste inhoud om daarmee de HTML pagina's aan te maken.

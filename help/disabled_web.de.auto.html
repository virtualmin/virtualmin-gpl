HTML der Website deaktiviert
In diesem Feld können Sie die Meldung anpassen, die beim Herstellen einer Verbindung zu einer Website für einen deaktivierten virtuellen Server angezeigt wird. Die Standardnachricht lautet einfach Website deaktiviert . Sie können typische Ersetzungen wie ${DOM} und ${USER} verwenden, um Details zur deaktivierten Domain abzurufen, ${DISABLED_WHY}, um den Grund für die Deaktivierung der Domain abzurufen, und ${DISABLED_TIME}, um den Zeitpunkt abzurufen, zu dem die Deaktivierung erfolgt geschah.

Запросить сертификат Let's Encrypt при создании домена?
Если установлено значение «Да» , Virtualmin попытается запросить действительный сертификат SSL Let's Encrypt для новых виртуальных серверов. Это будет успешным только в том случае, если у них есть доменное имя, которое разрешается извне вашей системы, чтобы его можно было найти службой Let's Encrypt.

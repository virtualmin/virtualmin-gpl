Lien personnalisé
Chaque lien personnalisé possède les attributs suivants que vous pouvez configurer sur ce formulaire:

Titre du lien
Le texte du lien. Cela peut inclure des substitutions comme $DOM et $USER, qui sont remplacées par des détails du domaine sélectionné.
URL de destination
URL complète vers laquelle pointe le lien. Encore une fois, des substitutions comme $DOM peuvent être utilisées.
Ouvrir dans la fenêtre
Si Identique est sélectionné, le lien s'ouvrira dans le cadre de droite. Si Nouveau est choisi, il s'ouvrira dans une fenêtre de navigateur de niveau supérieur séparée.
Catégorie
Si vous avez défini des catégories personnalisées pour les liens, vous pouvez sélectionner une catégorie pour chaque lien dans cette colonne.
Les domaines et les utilisateurs vers lesquels les liens semblent être contrôlés peuvent également être contrôlés à l'aide des champs suivants sur cette page:

Types d'utilisateurs Virtualmin
Pour limiter les liens vers l'administrateur principal, les propriétaires de domaine ou les revendeurs Virtualmin, cochez les cases appropriées dans ce champ.
Serveurs virtuels à partir du modèle
Ce n'est que lorsqu'un serveur virtuel créé à l'aide du modèle sélectionné est choisi dans le menu de gauche que ce lien sera visible.
Serveurs virtuels avec fonctionnalité
Ce n'est que lorsqu'un serveur virtuel avec la fonctionnalité sélectionnée est choisi que le lien apparaît.

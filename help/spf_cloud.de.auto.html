Hosting für DNS-Einträge
Wenn Sie andere Cloud-DNS-Anbieter konfiguriert haben, können Sie mit dieser Option das DNS-Hosting für diese Domäne zwischen ihnen oder vom lokalen Hosting in die Cloud verschieben. Beachten Sie, dass dies möglicherweise eine Aktualisierung dieser Nameserver-IP-Adressen bei Ihrem DNS-Registrar erforderlich macht, nachdem Sie den Umzug vorgenommen haben.

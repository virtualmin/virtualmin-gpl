Esegui script CGI usando
Quando il server fcgiwrap è installato, questa opzione consente di selezionare se verrà sempre utilizzato per eseguire script CGI per nuovi domini. In caso contrario, verrà utilizzato suexec se installato e configurato correttamente.

Fcgiwrap è generalmente più veloce nell'esecuzione di script CGI, ma richiede che ogni server virtuale esegua il proprio processo server che consuma RAM aggiuntiva.

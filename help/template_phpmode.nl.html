Standaard PHP uitvoer mode
Deze optie bepaald of PHP scripts in nieuwe virtuele servers uitgevoerd moeten worden als de Apache web server gebruiker (normaal httpd of apache), of als de Unix gebruiker waar de virtuele server van is. De eerste optie is het snelst omdat er geen extra processen nodig zijn om de PHP code te starten. Maar het is minder veilig omdat scripts lees en schrijf rechten hebben als een gebruiker en niet als een domein eigenaar.

Als uw systeem het ondersteunt is de b>FCGId optie het beste omdat het PHP scripts snel uitvoert en als domein eigenaar werkt. In alle andere gevallen als uw systeem niet zwaar belast word en veiligheid tussen gebruikers geen probleem is dan is de CGI wrapper optie aanbevolen.

Nombre de host para el registro MX
De manera predeterminada, el registro MX de DNS (que controla dónde se entrega el correo electrónico) para los nuevos dominios será el correo. $DOMAIN, al que se le asignará la misma dirección IP que el sitio web del servidor virtual. Sin embargo, puede usar este campo para especificar un nombre de host del servidor de correo diferente; esto generalmente se usa para tener un registro MX consistente en todos los dominios. Sin embargo, asegúrese de ingresar un nombre de host que se resuelva en su sistema Virtualmin, de lo contrario, no se entregará el correo electrónico.

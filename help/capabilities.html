Can edit virtual server
Allows editing of virtual server features, its password and description.
Can manage users
Allows mailbox, FTP and database users to be viewed, edited, created and deleted.
Can manage aliases
Allows email aliases to be viewed, edited, created and deleted.
Can manage databases
Allows additional MySQL and PostgreSQL databases to be added to this virtual server or deleted from it.
Can install scripts
Allows third-party scripts to be installed into this virtual server.
Can change IP address
If this server has a private IP address, this option allows it to be changed.
Can managed SSL certificates
Allows the SSL certificate and key for this server to be uploaded or re-generated.
Can edit forwarding frame
For forwarding-only websites, this option allows the destination and HTML for the forwarding frame to be edited.
Can manage extra admins
Allows extra server owner administrator accounts to be viewed, created, edited and deleted.
Can configure spam and virus delivery
Allows spam and virus delivery destinations and scores to be edited.
Can select PHP versions
For domains with PHP run via CGI or fastCGI, allows per-directory PHP versions to be set.
Can edit email settings
Allows configuration of automatic BCCing, alias copy mode and other email-related optoins.
Can make backups
Allows owned virtual servers to be backed up to a remote FTP or SSH server.
Can schedule backups
Allows virtual servers to be backed up on a regular schedule.
Can restore backups
Allows home directory and database content of owned virtual servers to be restored.
Can select shared IPs
Allows globally defined shared IP addresses to be selected when creating virtual servers.
Can create catchall aliases
Allows creation of email aliases that forward all mail to a domain to a different domain or address.
Can use web page editor
Allows web pages to be edited via the Virtualmin UI.
Can edit database remote hosts
Allows entering of remote client systems that are allowed to connect to the domain's MySQL databases.
Can disable virtual servers
Allows servers to be temporarily disabled.
Can delete virtual servers
Allows owned servers to be completely deleted, including all sub-servers and aliases.

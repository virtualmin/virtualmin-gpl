Nombre de base de datos predeterminado
Virtualmin permite la creación de bases de datos y permisos muy flexibles. Este campo le permite especificar el nombre predeterminado de las bases de datos creadas como parte de los dominios creados a partir de esta plantilla. De manera predeterminada, Virtualmin usa la variable ${PREFIX}, que es la primera parte del nombre de dominio, y generalmente también será el nombre de inicio de sesión del propietario del dominio para Webmin. Puede ser útil cambiar esto, en algunas circunstancias para incluir un prefijo que indique la plantilla a partir de la cual se generó, o para incluir información adicional sobre la base de datos. Es posible combinar variables y cualquier carácter permitido en una base de datos PostgreSQL y / o MySQL (dependiendo de las bases de datos disponibles).

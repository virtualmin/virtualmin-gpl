Nome del database predefinito
Virtualmin consente la creazione e le autorizzazioni di database molto flessibili. Questo campo consente di specificare il nome predefinito dei database creati come parte dei domini creati da questo modello. Per impostazione predefinita, Virtualmin utilizza la variabile ${PREFIX}, che è la prima parte del nome di dominio, e sarà generalmente anche il nome di accesso del proprietario del dominio per Webmin. Può essere utile modificarlo, in alcune circostanze per includere un prefisso che indica il modello da cui è stato generato o per includere informazioni aggiuntive sul database. È possibile combinare variabili e caratteri consentiti in un database PostgreSQL e / o MySQL (a seconda dei database disponibili).

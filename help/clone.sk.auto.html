Klonovať virtuálny server
Táto stránka vám umožňuje duplikovať existujúci virtuálny server s novým názvom domény. Akýkoľvek webový obsah, záznamy DNS, poštové schránky, poštové aliasy, databázy a ďalšie nastavenia spojené s pôvodnou doménou budú podľa možnosti duplikované. Webový obsah sa však žiadnym spôsobom nezmení, takže ak virtuálny server obsahuje webovú aplikáciu, ktorá bola nakonfigurovaná na používanie databáz alebo domovského adresára starého servera, bude potrebné ju po klonovaní manuálne prekonfigurovať.

Pri duplikovaní virtuálneho servera najvyššej úrovne musíte tiež zadať nové používateľské meno a heslo, aby ste mohli vlastniť novú doménu. Ak má server súkromnú IP adresu, klonu sa automaticky pridelí nová IP.

Proces klonovania skopíruje všetky databázy spojené s pôvodnou doménou pod novými názvami a duplikuje ich obsah. Neaktualizuje však žiadne skripty nainštalované v pôvodnej doméne, ktoré sú nakonfigurované na používanie pôvodných databáz – to je potrebné vykonať manuálne.

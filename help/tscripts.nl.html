Template Script Installers
Deze pagina geeft u de mogelijkheid om 3e partij scripts te specificeren die mogen worden geinstalleerd wanneer er een nieuwe virtuele server word aangemaakt met deze template. Voor ieder script moet de volgende informatie worden ingevuld :

Script
De naam en versie van het te installeren script
URL pad
Een pad onder de virtuele server website om het script te installeren, zoals /phpmyadmin. Dit bepaald tevens de directory onder de ~/public_html waar de script files zijn opgeslagen.
DB naam
De naam van de database die het script gaat gebruiken om data in op te slaan. De opties zijn :
Geen
Dit moet geselecteerd worden voor scripts die geen database nodig hebben, zoals Gallery.
Hoofd database
De script's database zal de virtuele server primaire database zijn van het gekozen type.
Anders
Als u wilt dat het script zijn eigen prive database gebruikt, kan deze optie geselecteerd worden en het patroon voor de database kan worden ingevuld in het naastgelegen tekst veld. Bijvoorbeeld, u kunt invullen ${DB}_phpnuke. Als de database niet bestaat zal het worden aangemaakt als onderdeel van het script installatie proces.
DB type
Het type database wat word gebruikt voor de script data, indien dit is geselecteerd.

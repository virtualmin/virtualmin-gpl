Comando a ser executado antes de fazer alterações em um domínio
Se você deseja executar alguma ação antes de alterar um domínio (criação, alteração ou exclusão), pode especificar um comando para executar aqui. Se várias ações devem ser executadas, o comando a executar pode ser um script. O comando tem acesso a variáveis de ambiente especiais específicas do domínio, como VIRTUALSERVER_DOM, VIRTUALSERVER_IP, VIRTUALSERVER_USER, VIRTUALSERVER_HOME, VIRTUALSERVER_PASS e muito mais. Qualquer variável encontrada no arquivo / etc / webmin / servidor virtual / domínios / $DOM estará disponível como uma variável de ambiente.

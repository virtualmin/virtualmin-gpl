Malfil for e-post til nye postkasser
Hvis nye postbokser som opprettes automatisk skal sendes en melding, kan du angi den her. Det kan være standard, som kan redigeres ved å bruke e- postsiden New Mailbox i Virtualmin, eller du kan gi en annen tekstfil hvor som helst på systemet. Variabel substitusjon med følgende variabler: ${MAILBOX}, ${USER}, ${DOM}, ${FTP} og ${HOME}. Du kan også bruke de spesielle logikkblokkoperatørene $IF-MAIL ... $ENDIF-MAIL og $IF-WEB ... $ELSE-WEB ... {$ENDIF-WEB} for å la meldingen din lage en passende melding basert på funksjonene brukeren har tilgang til. Hvis du bruker DAV, kan blokken ${IF-VIRTUALMIN-DAV} .. ${ENDIF-VIRTUALMIN-DAV} brukes til å få en annen tekst til å vises, avhengig av om brukeren har DAV aktiv eller ikke.

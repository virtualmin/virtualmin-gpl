Instaladores de scripts de plantilla
Esta página le permite especificar scripts de terceros que se instalarán cuando se cree un nuevo servidor virtual utilizando esta plantilla. Para cada script, se debe ingresar la siguiente información:

Guión
El nombre y la versión de un script para instalar
Ruta URL
Una ruta debajo del sitio web del servidor virtual para instalar el script, como /phpmyadmin . Esto también determinará el directorio bajo ~/public_html que contiene los archivos del script.
Nombre de base de datos
El nombre de la base de datos que el script usará para almacenar datos. Las opciones son:
Ninguna
Esto debe seleccionarse para los scripts que no necesitan una base de datos, como Galería .
Base de datos principal
La base de datos del script será la base de datos primaria del servidor virtual del tipo elegido.
Otro
Si desea que este script use su propia base de datos privada, esta opción puede seleccionarse y el patrón para una base de datos ingresado en el campo de texto adyacente. Por ejemplo, puede ingresar ${DB}_phpnuke . Si la base de datos resultante no existe, se creará como parte del proceso de instalación del script.
Tipo de base de datos
El tipo de uso de la base de datos para los datos de los scripts, si hay alguno seleccionado.

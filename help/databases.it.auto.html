Modifica database
Questa pagina elenca tutti i database di proprietà del server virtuale e consente di crearne di nuovi, fino al limite consentito per l'utente connesso. È anche possibile eliminare uno o più database.

Per visualizzare le tabelle e i dati di un database, clicca sul pulsante "Gestisci". Se phpMyAdmin è installato per questo server virtuale, vedrai anche il pulsante "Gestisci in phpMyAdmin". Se il server virtuale è un sottoserver e ha una propria installazione di phpMyAdmin, verrà utilizzata quest'ultima al posto di quella del server principale. Se non è presente un'installazione per server, ma è stata impostata un'istanza predefinita di phpMyAdmin, verrà utilizzata quell'istanza condivisa.

Per visualizzare ulteriori dettagli su un database, tra cui le sue dimensioni e il numero di tabelle, fare clic sul suo nome.

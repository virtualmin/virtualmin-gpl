Modifica database
Questa pagina elenca tutti i database di proprietà del server virtuale e consente di crearne di nuovi, fino al limite consentito per l'utente connesso. È anche possibile eliminare uno o più database.

Per visualizzare le tabelle e i dati di un database, clicca sul pulsante "Gestisci". Se phpMyAdmin è installato su questo server virtuale, vedrai anche il pulsante "Gestisci in phpMyAdmin". Se phpMyAdmin è installato su un sottoserver, verrà utilizzata l'installazione di phpMyAdmin del sottoserver anziché quella del server padre.

Per visualizzare ulteriori dettagli su un database, tra cui le sue dimensioni e il numero di tabelle, fare clic sul suo nome.

Préfixe pour des bases de données supplémentaires
Par défaut, Virtualmin permettra aux propriétaires de serveurs de créer des bases de données supplémentaires avec n'importe quel nom. Cependant, ce champ peut être utilisé pour appliquer un préfixe. Une bonne valeur est ${PREFIX} _, ce qui forcera le nom de la base de données à commencer par la première partie du nom de domaine.

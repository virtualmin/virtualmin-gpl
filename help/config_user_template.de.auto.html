Vorlagendatei für E-Mails an neue Postfächer
Wenn neu erstellte Postfächer automatisch eine Nachricht gesendet werden sollen, können Sie diese hier angeben. Dies kann die Standardeinstellung sein, die über die E-Mail- Seite Neue Mailbox von Virtualmin bearbeitet werden kann, oder Sie können eine beliebige Textdatei an einer beliebigen Stelle im System bereitstellen. Variablensubstitution mit den folgenden Variablen: ${MAILBOX}, ${USER}, ${DOM}, ${FTP} und ${HOME}. Sie können auch die speziellen Logikblockoperatoren $IF-MAIL ... $ENDIF-MAIL und $IF-WEB ... $ELSE-WEB ... {$ENDIF-WEB} verwenden, damit Ihre Nachricht eine entsprechende Nachricht erstellen kann basierend auf den Funktionen, auf die der Benutzer Zugriff hat. Wenn Sie DAV verwenden, kann der Block ${IF-VIRTUALMIN-DAV} .. ${ENDIF-VIRTUALMIN-DAV} verwendet werden, um unterschiedlichen Text anzuzeigen, je nachdem, ob der Benutzer DAV aktiv hat oder nicht.

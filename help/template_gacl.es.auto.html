Configuraciones de selección de usuario y directorio
Webmin proporciona una ventana emergente estándar de usuario y selector de directorio, que se puede configurar para proporcionar acceso a algunos o todos los usuarios o directorios. Los valores predeterminados generalmente son relativamente restrictivos y solo permiten el acceso a los usuarios dentro del dominio que pertenece a la cuenta del propietario del dominio y a los directorios y archivos que pertenecen al grupo de propietarios del dominio. Si desea configurar explícitamente estas opciones, seleccione "Configuración a continuación .." Tenga cuidado con estas opciones, ya que es muy fácil otorgar privilegios que no deberían otorgarse a un usuario no confiable. La sustitución de variables se produce para las siguientes opciones relacionadas, por lo que es posible especificar ${USER} para seleccionar el usuario que posee el dominio o ${HOME} para seleccionar el directorio de inicio del propietario del dominio, por ejemplo.

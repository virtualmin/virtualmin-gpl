Modelo de servidor
Um modelo de servidor define as configurações padrão para novos domínios ou para os recursos adicionados aos domínios existentes. Quase todos os campos e caixas de texto abaixo permitem que você use variáveis que serão substituídas com base na configuração do servidor virtual no momento em que o modelo é usado. Algumas das variáveis mais úteis são:

${DOM} O nome do domínio, como foo.com
${PREFIX} A primeira parte do nome de domínio, como foo
${USER} O usuário que possui o domínio, como foo
${IP} O endereço IP atribuído ao servidor virtual, como 192.168.1.1
${HOME} O diretório inicial do usuário do domínio, como /home/foo
${UID} O ID do usuário do sistema do usuário proprietário do domínio, como 502 .
Além disso, blocos condicionais como $IF-MAIL ... $ENDIF-MAIL e $IF-WEB ... $ELSE-WEB ... {$ENDIF-WEB} podem ser usados.

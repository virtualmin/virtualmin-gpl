Prihlásenie SSH/Telnet
Táto možnosť určuje, či môže používateľ vlastníka domény používať modul SSH/Telnet Login Webmin. Tento modul je Java applet, ktorý sa prihlasuje na server pomocou SSH alebo Telnetu. Nepracuje správne so všetkými prehliadačmi kvôli nekompatibilite Java. Upozorňujeme, že zakázanie tohto nezabráni vlastníkovi domény v prístupe SSH na server, ale iba v prístupe k modulu SSH/Telnet Login Webmin.

Usar como predeterminado del sistema phpMyAdmin
Al habilitarse, esta instalación de phpMyAdmin se establece como predeterminada para todo el sistema. Se mostrará el botón "Administrar en phpMyAdmin" en la página "Editar bases de datos" para los dominios que no tengan su propia instalación de phpMyAdmin.

El phpMyAdmin que se utiliza depende de lo que esté instalado:

  • Si el subservidor tiene su propia instalación de phpMyAdmin, se utiliza esa instalación
  • De lo contrario, si el servidor virtual de nivel superior tiene instalado phpMyAdmin, esa instalación se utiliza tanto para el servidor de nivel superior como para cualquier subservidor que no tenga su propio phpMyAdmin.
  • De lo contrario, se utilizará la instalación predeterminada de phpMyAdmin del sistema.

Esto puede ser útil cuando desea una instancia compartida de phpMyAdmin (por ejemplo, bajo un nombre de host dedicado como pma.example.tld ) y al mismo tiempo permitir instalaciones por servidor virtual cuando sea necesario.

Tenga cuidado: solo use un phpMyAdmin compartido "predeterminado del sistema" con un nombre de host dedicado si confía plenamente en el servidor virtual que lo aloja. Si otros servidores virtuales no tienen instalado su propio phpMyAdmin, usarán esta instancia predeterminada, lo que puede implicar que sus credenciales de inicio de sesión de MySQL se le envíen para iniciar sesión. Si el servidor virtual que lo aloja está comprometido, mal configurado o controlado por alguien en quien no confía plenamente, esas credenciales podrían quedar expuestas.

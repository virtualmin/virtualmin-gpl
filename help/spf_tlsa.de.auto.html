TLSA-Einträge aktiviert?
Wenn aktiviert, erstellt Virtualmin TLSA-DNS-Einträge, die zum Veröffentlichen von SSL-Zertifikaten für diesen virtuellen Server verwendet werden. Clients können dann überprüfen, ob das erwartete Zertifikat präsentiert wird, was eine zusätzliche Abwehr gegen Man-in-the-Middle-Angriffe bietet.

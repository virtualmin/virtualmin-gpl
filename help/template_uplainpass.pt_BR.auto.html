Armazenar senhas em texto simples?
Quando esta opção é definida como Sim , o Virtualmin armazenará as senhas em texto não criptografado para proprietários de domínio e caixas de correio. Isso permite que eles sejam exibidos na IU, mas adiciona um risco potencial de segurança se o sistema for comprometido. Selecionar Não fará com que todos os novos servidores virtuais criados a partir deste modelo armazenem apenas senhas com hash unilateral.

Como os aplicativos da web precisam armazenar as senhas do banco de dados em texto simples, a senha do banco de dados para um usuário será definida como uma nova senha aleatória se o Virtualmin estiver configurado para não armazenar senhas em texto simples. Você pode ver essa senha em Editar bancos de dados> Senhas .

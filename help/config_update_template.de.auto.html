Vorlagendatei für E-Mails an aktualisierte Postfächer
Wenn Postfächer bei der Aktualisierung automatisch eine Nachricht erhalten sollen, können Sie diese hier angeben. Dies kann die Standardeinstellung sein, die über die Seite " Aktualisierte Postfach-E-Mail " von Virtualmin bearbeitet werden kann, oder Sie können eine andere Textdatei an einer beliebigen Stelle im System bereitstellen. Variablensubstitution mit den folgenden Variablen: ${MAILBOX}, ${USER}, ${DOM}, ${FTP} und ${HOME}. Sie können auch die speziellen Logikblockoperatoren $IF-MAIL ... $ENDIF-MAIL und $IF-WEB ... $ELSE-WEB ... {$ENDIF-WEB} verwenden, damit Ihre Nachricht eine entsprechende Nachricht erstellen kann basierend auf den Funktionen, auf die der Benutzer Zugriff hat.

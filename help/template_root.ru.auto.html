Корневой каталог в окне выбора файлов
Этот параметр указывает, какой каталог будет отображаться средством выбора файлов при первом открытии. При наличии соответствующих разрешений пользователь может затем перейти к другим каталогам. Подстановка переменной может быть выполнена с использованием переменной ${HOME}, чтобы указать домашний каталог нового домена.

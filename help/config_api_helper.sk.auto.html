Cesta na vytvorenie pomocného príkazu API
V predvolenom nastavení Virtualmin vytvorí pomocný skript API príkazového riadka virtualmin /usr/sbin alebo /usr/local/sbin , v závislosti od toho, do ktorého systému je možné zapisovať. Toto konfiguračné pole však možno použiť na výber iného umiestnenia, napríklad /opt/bin/virtualmin ktoré je vhodné pre vašu lokalitu.

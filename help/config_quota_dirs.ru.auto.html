Дополнительные каталоги для поиска файлов в
Это поле можно использовать для перечисления каталогов, в которых следует выполнять поиск при отображении страницы использования диска, чтобы показать использование диска для виртуального сервера, даже если он находится за пределами домашнего каталога или файловой системы. В пути можно использовать стандартные замены, такие как $DOM и $USER.

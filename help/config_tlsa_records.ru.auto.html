Создать записи DNS TLSA для сертификатов SSL?
Если этот параметр включен, Virtualmin добавит DNS-запись TLSA для всех имен хостов, использующих сертификаты SSL для вновь созданных виртуальных серверов. Это позволяет клиентам подтверждать, что сертификат SSL, представленный вашему веб-серверу, совпадает с сертификатом, размещенным в вашей системе, что предотвращает определенные атаки типа «злоумышленник в середине».

Sottodirectory home
I nuovi domini (che non appartengono ai proprietari di domini esistenti) risiederanno in una sottodirectory all'interno della home directory sopra specificata. Per impostazione predefinita, il nome della directory corrisponderà al nome utente e risiederà direttamente nella directory di base. Puoi anche specificare un nome di directory speciale, se, ad esempio, desideri mantenere separati gli utenti di sistema standard dagli utenti degli account di dominio. In tal caso, è possibile posizionare i domini Virtualmin in "/ home / domains / $USER".

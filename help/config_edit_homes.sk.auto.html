Je možné vybrať domovské adresáre pre používateľov?
Ak je táto možnosť povolená, vlastník domény sa môže rozhodnúť poskytnúť používateľom iný domovský adresár, než je predvolený. Môže byť užitočné, ak si vlastníci domén môžu nainštalovať ďalšie služby, ako napríklad aplikačný server (napr. Zope, Webrick atď.), a chceli by mať možnosť používať iného používateľa pre server aplikácie. Vo všeobecnosti budú tento druh flexibility potrebovať iba používatelia, ktorí majú relatívne vysokú úroveň dôveryhodnosti.

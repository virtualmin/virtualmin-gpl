Exécuter des scripts CGI en utilisant
Lorsque le serveur fcgiwrap est installé, cette option vous permet de sélectionner s'il sera toujours utilisé pour exécuter des scripts CGI pour les nouveaux domaines. Sinon, suexec sera utilisé s'il est installé et configuré correctement.

FCGIwrap est généralement plus rapide pour exécuter des scripts CGI, mais nécessite que chaque serveur virtuel exécute son propre processus serveur qui consomme de la RAM supplémentaire.

Spouštějte skripty CGI pomocí
Když je nainstalován server fcgiwrap , tato možnost vám umožňuje vybrat, zda bude vždy používán ke spouštění skriptů CGI pro nové domény. V opačném případě bude suexec použit, pokud je nainstalován a nakonfigurován správně.

FCGIwrap je obecně rychlejší pro spouštění CGI skriptů, ale vyžaduje, aby každý virtuální server spouštěl svůj vlastní serverový proces, který spotřebovává další RAM.

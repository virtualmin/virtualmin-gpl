Anfrage Lassen Sie uns das Zertifikat bei der Erstellung der Domain verschlüsseln?
Bei der Einstellung Ja versucht Virtualmin, ein gültiges Let's Encrypt SSL-Zertifikat für neue virtuelle Server anzufordern. Dies ist nur erfolgreich, wenn sie einen Domainnamen haben, der von außerhalb Ihres Systems aufgelöst werden kann, damit er vom Let's Encrypt-Dienst nachgeschlagen werden kann.

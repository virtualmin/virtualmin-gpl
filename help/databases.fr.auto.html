Modifier les bases de données
Cette page répertorie toutes les bases de données appartenant au serveur virtuel et vous permet d'en créer de nouvelles, dans la limite autorisée pour l'utilisateur connecté. Vous pouvez également supprimer une ou plusieurs bases de données.

Pour consulter les tables et les données d'une base de données, cliquez sur le bouton « Gérer ». Si phpMyAdmin est installé sur ce serveur virtuel, un bouton « Gérer dans phpMyAdmin » s'affichera également. Si phpMyAdmin est installé sur un sous-serveur, c'est l'installation de ce dernier qui sera utilisée.

Pour afficher des informations supplémentaires sur une base de données, notamment sa taille et le nombre de tables, cliquez sur son nom.

Modifier les bases de données
Cette page répertorie toutes les bases de données appartenant au serveur virtuel et vous permet d'en créer de nouvelles, dans la limite autorisée pour l'utilisateur connecté. Vous pouvez également supprimer une ou plusieurs bases de données.

Pour consulter les tables et les données d'une base de données, cliquez sur le bouton « Gérer ». Si phpMyAdmin est installé sur ce serveur virtuel, un bouton « Gérer dans phpMyAdmin » s'affichera également. Si le serveur virtuel est un sous-serveur et qu'il possède sa propre installation de phpMyAdmin, c'est cette dernière qui sera utilisée. En l'absence d'installation spécifique au serveur, mais si une instance système de phpMyAdmin par défaut est définie, c'est cette instance partagée qui sera utilisée.

Pour afficher plus de détails sur une base de données, notamment sa taille et le nombre de tables, cliquez sur son nom.

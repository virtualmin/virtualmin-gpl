Plantilla de servidor
Una plantilla de servidor define la configuración predeterminada para nuevos dominios, o para características agregadas a dominios existentes. Casi todos los campos y cuadros de texto a continuación le permiten usar variables que serán sustituidas en función de la configuración del servidor virtual en el momento en que se usa la plantilla. Algunas de las variables más útiles son:

${DOM} El nombre de dominio, como foo.com
${PREFIX} La primera parte del nombre de dominio, como foo
${USER} El usuario que posee el dominio, como foo
${IP} La dirección IP asignada al servidor virtual, como 192.168.1.1
${HOME} El directorio de inicio del usuario del dominio, como /home/foo
${UID} El ID de usuario del sistema del usuario propietario del dominio, como 502 .
Además, se pueden usar bloques condicionales como $IF-MAIL ... $ENDIF-MAIL y $IF-WEB ... $ELSE-WEB ... {$ENDIF-WEB} .

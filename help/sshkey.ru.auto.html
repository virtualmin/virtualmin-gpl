Открытый ключ SSH
Это поле можно использовать для предоставления доступа по SSH, аутентифицированного с помощью открытого ключа, пользователю Unix нового виртуального сервера. Возможные варианты:

Нет ключа по умолчанию
Не настраивайте SSH-ключ. Разрешены только входы с паролем.

Генерация публичных и приватных ключей
Создайте новую пару ключей SSH для этого виртуального сервера и разрешите вход с использованием этого ключа. Владелец сервера должен скопировать закрытый ключ после создания.

Используйте открытый ключ ниже
Вход по SSH будет разрешен с использованием закрытого ключа, который соответствует открытому ключу, введенному в текстовое поле ниже.

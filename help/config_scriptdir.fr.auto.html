Modèle de répertoire de script par défaut
Par défaut, Virtualmin installe de nouveaux scripts dans un sous-répertoire portant le même nom que le code de script, comme phpadmin . Ce champ vous permet de saisir un modèle pour un répertoire par défaut différent, comme scripts/${SCRIPTNAME} . Toutes les variables de modèle habituelles comme ${DOM} et ${USER} peuvent également être utilisées.

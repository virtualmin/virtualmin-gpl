Datenbanken bearbeiten
Diese Seite listet alle Datenbanken des virtuellen Servers auf und ermöglicht es Ihnen, neue Datenbanken bis zum für den angemeldeten Benutzer festgelegten Limit zu erstellen. Sie können auch eine oder mehrere Datenbanken löschen.

Um die Tabellen und Daten einer Datenbank anzuzeigen, klicken Sie auf die Schaltfläche „Verwalten“. Wenn phpMyAdmin auf diesem virtuellen Server installiert ist, wird Ihnen zusätzlich die Schaltfläche „In phpMyAdmin verwalten“ angezeigt. Falls es sich bei dem virtuellen Server um einen Subserver mit eigener phpMyAdmin-Installation handelt, wird diese anstelle der Installation des übergeordneten Servers verwendet. Ist keine serverseitige Installation vorhanden, sondern eine systemweite Standardinstallation von phpMyAdmin festgelegt, wird diese gemeinsam genutzte Instanz verwendet.

Um weitere Details zu einer Datenbank anzuzeigen, einschließlich ihrer Größe und der Anzahl der Tabellen, klicken Sie auf ihren Namen.

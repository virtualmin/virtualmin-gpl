Zdieľané adresy IP
Primárnym účelom tejto stránky je zadanie dodatočných IP adries, ktoré môžu používať virtuálne servery, ktoré nemajú svoje súkromné IP adresy. Každá zadaná IP musí byť v systéme už aktívna (čo je možné vykonať v module Webmin Network Configuration) a nemôže byť použitá žiadnym iným predajcom alebo súkromným IP virtuálnym serverom.

Link personalizzato
Ogni collegamento personalizzato ha i seguenti attributi che è possibile configurare in questo modulo:

Titolo del link
Il testo del link. Ciò può includere sostituzioni come $DOM e $USER, che vengono sostituite dai dettagli del dominio selezionato.
URL di destinazione
L'URL completo a cui punta il collegamento. Ancora una volta, è possibile utilizzare sostituzioni come $DOM.
Apri nella finestra
Se è selezionato Stesso , il collegamento si aprirà nel riquadro di destra. Se si seleziona Nuovo , si aprirà in una finestra del browser di livello superiore separata.
Categoria
Se sono state definite categorie personalizzate per i collegamenti, è possibile selezionare una categoria per ciascun collegamento in questa colonna.
I domini e gli utenti a cui appaiono i collegamenti possono anche essere controllati utilizzando i seguenti campi in questa pagina:

Tipi di utenti Virtualmin
Per limitare i collegamenti all'amministratore principale, ai proprietari del dominio o ai rivenditori Virtualmin, selezionare le caselle appropriate in questo campo.
Server virtuali dal modello
Solo quando un server virtuale creato usando il modello selezionato viene scelto dal menu a sinistra questo link sarà visibile.
Server virtuali con funzionalità
Solo quando viene scelto un server virtuale con la funzione selezionata verrà visualizzato il collegamento.

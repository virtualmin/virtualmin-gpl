Estilo de nome de domínio em nomes de usuário
Esta opção determina se o nome de domínio usado nos nomes de usuário que contêm o domínio será o nome completo do domínio, ou seja, "virtualmin.com", apenas a primeira parte, ou seja, "virtualmin", ou o nome do usuário / grupo do domínio. Por padrão, o nome do grupo é usado para novos servidores de nível superior e a primeira parte do domínio é usada para sub-servidores.

Em alguns ambientes, pode ser mais fácil para os usuários entender um nome de usuário como "joeuser@virtualmin.com" para seu nome de login POP / IMAP ou FTP do que "joeuser.virtualmin".

Este campo também determina como o prefixo de um novo servidor virtual é calculado. As opções são:


Nome de domínio completo
O prefixo é sempre igual ao nome do domínio.

Nome de usuário ou primeira parte do domínio
O prefixo é o mesmo que o nome do grupo Unix do servidor virtual, se for um servidor virtual de nível superior. Caso contrário, a seguinte opção se aplica.

Primeira parte do domínio
O prefixo é a primeira parte do domínio, a menos que outro servidor virtual já esteja usando esse prefixo. Portanto, para o domínio foo.com.au , o prefixo seria (em ordem de preferência) foo , foo.com.au ou foo.com

Padrão personalizado de expressão regular
O prefixo será gerado automaticamente com base no padrão de expressão regular personalizado, como u-[0-9]{4} ou [az]-[0-9]{2} ou user-[0-9]{2}-${year}-${month}-${day} que produzirá u-1234 ou x-89 ou user-99-2021-12-25 acordo.



Ele também determina o nome de usuário gerado automaticamente para um domínio de nível superior, que será igual ao prefixo.

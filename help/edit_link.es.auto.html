Enlace personalizado
Cada enlace personalizado tiene los siguientes atributos que puede configurar en este formulario:

Título del enlace
El texto del enlace. Esto puede incluir sustituciones como $DOM y $USER, que se reemplazan por detalles del dominio seleccionado.
URL de destino
La URL completa a la que apunta el enlace. Nuevamente, se pueden usar sustituciones como $DOM.
Abrir en ventana
Si se selecciona Mismo , el enlace se abrirá en el marco del lado derecho. Si se elige Nuevo , se abrirá en una ventana separada del navegador de nivel superior.
Categoría
Si ha definido categorías personalizadas para enlaces, puede seleccionar una categoría para cada enlace en esta columna.
Los dominios y usuarios a los que parecen vincularse también se pueden controlar utilizando los siguientes campos en esta página:

Tipos de usuario de Virtualmin
Para limitar los enlaces al administrador maestro, los propietarios del dominio o los revendedores de Virtualmin, marque las casillas correspondientes en este campo.
Servidores virtuales desde plantilla
Solo cuando se elija un servidor virtual creado con la plantilla seleccionada en el menú de la izquierda, este enlace será visible.
Servidores virtuales con función
Solo cuando se elija un servidor virtual con la función seleccionada, aparecerá el enlace.

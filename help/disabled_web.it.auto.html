HTML del sito Web disabilitato
Questo campo può essere utilizzato per personalizzare il messaggio che appare quando ci si collega a un sito Web per un server virtuale disabilitato. Il messaggio predefinito indica semplicemente il sito Web disabilitato . Puoi utilizzare sostituzioni tipiche come ${DOM} e ${USER} per ottenere i dettagli del dominio disabilitato, ${DISABLED_WHY} per ottenere il motivo per cui il dominio è stato disabilitato e ${DISABLED_TIME} per ottenere l'ora in cui disabilita è stato fatto.

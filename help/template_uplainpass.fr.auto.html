Stocker les mots de passe en clair?
Lorsque cette option est définie sur Oui , Virtualmin stockera les mots de passe en texte clair pour les propriétaires de domaine et les boîtes aux lettres. Cela leur permet d'être affichés dans l'interface utilisateur, mais ajoute un risque de sécurité potentiel si votre système est compromis. Si vous sélectionnez Non , tous les nouveaux serveurs virtuels créés à partir de ce modèle ne stockeront que des mots de passe hachés unidirectionnels à la place.

Étant donné que les applications Web doivent stocker les mots de passe de base de données en texte brut, le mot de passe de la base de données d'un utilisateur sera défini sur un nouveau mot de passe aléatoire si Virtualmin est configuré pour ne pas stocker les mots de passe en texte clair. Vous pouvez afficher ce mot de passe en allant dans Modifier les bases de données> Mots de passe .

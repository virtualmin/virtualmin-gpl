Memorizzare le password in chiaro?
Quando questa opzione è impostata su Sì , Virtualmin memorizzerà le password in testo non crittografato per i proprietari di dominio e le cassette postali. Ciò consente di visualizzarli nell'interfaccia utente, ma aggiunge un potenziale rischio per la sicurezza se il sistema è compromesso. Selezionando No , tutti i nuovi server virtuali creati da questo modello memorizzeranno invece solo le password con hash unidirezionale.

Poiché le applicazioni Web devono memorizzare le password del database in testo normale, la password del database per un utente verrà impostata su una nuova password casuale se Virtualmin è impostato per non memorizzare le password in chiaro. È possibile visualizzare questa password andando su Modifica database> Password .

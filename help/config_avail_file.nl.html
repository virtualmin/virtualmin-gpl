File Manager (alleen de home directory)
De File Manager is een geavanceerd Java applet die een dubbel overzicht paneel heeft om het file systeem te bekijken op de server. Het is 1 van de meest krachtige opties van Webmin waarmee zelfs onervaren gebruikers files kunnen beheren, of directory's en permissies, tekst files kunnen bewerken, files kunnen uploaden en downloaden en nog veel meer. Standaard ziet de domein eigenaar alleen zijn eigen directory en subdirectory's.

Файл шаблона для электронной почты на обновленные почтовые ящики
Если при обновлении почтовым ящикам автоматически отправляется сообщение, вы можете указать его здесь. Это может быть значение по умолчанию, которое можно редактировать с помощью обновленной страницы электронной почты почтового ящика Virtualmin, или вы можете предоставить другой текстовый файл в любом месте системы. Подстановка переменных с использованием следующих переменных: ${MAILBOX}, ${USER}, ${DOM}, ${FTP} и ${HOME}. Вы также можете использовать специальные операторы логических блоков $IF-MAIL ... $ENDIF-MAIL и $IF-WEB ... $ELSE-WEB ... {$ENDIF-WEB}, чтобы ваше сообщение могло создать соответствующее сообщение. на основе функций, к которым пользователь имеет доступ.

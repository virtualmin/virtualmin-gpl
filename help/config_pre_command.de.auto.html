Befehl zum Ausführen, bevor Änderungen an einer Domäne vorgenommen werden
Wenn Sie vor dem Ändern einer Domäne (Erstellen, Ändern oder Löschen) eine Aktion ausführen möchten, können Sie hier einen Befehl angeben, der ausgeführt werden soll. Wenn mehrere Aktionen ausgeführt werden sollen, kann der auszuführende Befehl ein Skript sein. Der Befehl hat Zugriff auf spezielle umgebungsspezifische Umgebungsvariablen wie VIRTUALSERVER_DOM, VIRTUALSERVER_IP, VIRTUALSERVER_USER, VIRTUALSERVER_HOME, VIRTUALSERVER_PASS und mehr. Jede Variable in der Datei / etc / webmin / virtual-server / domain / $DOM ist als Umgebungsvariable verfügbar.

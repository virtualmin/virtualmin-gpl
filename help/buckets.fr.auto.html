Seaux Amazon S3

Cette page répertorie tous les compartiments Amazon S3 appartenant à des comptes S3 que Virtualmin est configuré pour utiliser sur la page Fournisseurs de stockage cloud. Les compartiments peuvent également être créés et modifiés à partir de cette page, et les règles de cycle de vie de contrôle d'accès et de suppression automatique qui s'appliquent aux fichiers qu'ils contiennent sont modifiées.

Le processus de sauvegarde de Virtualmin créera automatiquement de nouveaux compartiments selon les besoins, mais dans l'emplacement de stockage S3 par défaut. La création d'un compartiment à partir de cette page vous permet de sélectionner un emplacement de stockage plus proche de votre système Virtualmin.

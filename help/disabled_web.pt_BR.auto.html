HTML do site desativado
Este campo pode ser usado para personalizar a mensagem que aparece ao conectar-se a um site para um servidor virtual desabilitado. A mensagem padrão simplesmente indica Site desativado . Você pode usar substituições típicas como ${DOM} e ${USER} para obter detalhes do domínio que está sendo desativado, ${DISABLED_WHY} para obter o motivo pelo qual o domínio foi desativado e ${DISABLED_TIME} para obter o horário em que a desativação foi desativada. foi feito.

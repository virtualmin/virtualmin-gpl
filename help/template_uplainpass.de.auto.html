Klartext-Passwörter speichern?
Wenn diese Option auf Ja gesetzt ist , speichert Virtualmin die Klartextkennwörter für Domänenbesitzer und Postfächer. Dadurch können sie in der Benutzeroberfläche angezeigt werden, es besteht jedoch ein potenzielles Sicherheitsrisiko, wenn Ihr System gefährdet ist. Wenn Sie Nein auswählen, werden in allen neuen virtuellen Servern, die aus dieser Vorlage erstellt wurden, stattdessen nur Einweg-Hash-Kennwörter gespeichert.

Da Webanwendungen Datenbankkennwörter im Klartext speichern müssen, wird das Datenbankkennwort für einen Benutzer auf ein neues, zufälliges Kennwort festgelegt, wenn Virtualmin keine Klartextkennwörter speichert. Sie können dieses Kennwort anzeigen, indem Sie auf Datenbanken bearbeiten> Kennwörter klicken .

Instaladores de script de modelo
Esta página permite especificar scripts de terceiros a serem instalados quando um novo servidor virtual for criado usando este modelo. Para cada script, as seguintes informações devem ser inseridas:

Roteiro
O nome e a versão de um script para instalar
Caminho do URL
Um caminho no site do servidor virtual para instalar o script, como /phpmyadmin . Isso também determinará o diretório em ~/public_html que contém os arquivos do script.
Nome do banco de dados
O nome do banco de dados que o script usará para armazenar dados. As opções são:
Nenhum
Isso deve ser selecionado para scripts que não precisam de um banco de dados, como o Gallery .
Base de dados principal
O banco de dados do script será o banco de dados principal do servidor virtual do tipo escolhido.
De outros
Se você desejar que esse script use seu próprio banco de dados privado, essa opção poderá ser selecionada e o padrão para um banco de dados inserido no campo de texto adjacente. Por exemplo, você pode digitar $ {DB} _phpnuke . Se o banco de dados resultante não existir, ele será criado como parte do processo de instalação do script.
Tipo de banco de dados
O tipo de banco de dados usado para os dados dos scripts, se houver algum selecionado.

Tu je možné použiť nasledujúce náhrady. Pri použití šablóny budú nahradené skutočnými hodnotami z domény:

${DOM} Názov domény, napríklad foo.com
${IP} IP adresa virtuálneho servera, napríklad 1.2.3.4
${DNS_IP} Externá IP adresa virtuálneho servera použitá v záznamoch DNS
${USER} Administračné prihlásenie servera, ako napríklad foo
${GROUP} Unixová skupina pre tento virtuálny server.
${UID} and ${GID} ID užívateľa Unix servera a ID skupiny
${HOME} Domovský adresár servera, napríklad /home/foo
${EMAILTO} Kontaktná e-mailová adresa virtuálneho servera
${PARENT} Ak ide o podserver, jedinečné ID nadradenej domény.
${ALIAS} Ak ide o alias, jedinečné ID cieľovej domény.
${ID} Jedinečné číselné ID pre túto doménu.
${CREATED} Časová pečiatka, v ktorej bola táto doména vytvorená.
${DB} Názov databázy MySQL alebo PostgreSQL.
Okrem toho možno použiť podmienené bloky ako ${IF-MAIL}...${ENDIF-MAIL} a ${IF-WEB}...${ELSE-WEB}...{$ENDIF-WEB} .

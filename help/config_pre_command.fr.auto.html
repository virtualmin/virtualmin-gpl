Commande à exécuter avant d'apporter des modifications à un domaine
Si vous souhaitez effectuer une action avant de modifier un domaine (création, modification ou suppression), vous pouvez spécifier une commande à exécuter ici. Si plusieurs actions doivent être effectuées, la commande à exécuter peut être un script. La commande a accès à des variables d'environnement spéciales spécifiques au domaine, telles que VIRTUALSERVER_DOM, VIRTUALSERVER_IP, VIRTUALSERVER_USER, VIRTUALSERVER_HOME, VIRTUALSERVER_PASS, etc. Toute variable trouvée dans le fichier / etc / webmin / virtual-server / domaines / $DOM sera disponible en tant que variable d'environnement.

Hostname filter
By default, redirects apply to any hostname used to access the website. Available options are:
  • Any hostname to match all hostnames.
  • Selected hostname to choose one detected website hostname.
  • Specify hostname or pattern to type a custom value.
When using custom input:
  • Plain hostnames (like www.example.com) are matched exactly.
  • Simple wildcard patterns using * and ? (like *.example.com or mail?.example.com) are automatically converted to a regular expression.
  • Other non-space patterns are treated as regular expressions.

Because wildcard patterns are stored internally as regular expressions, re-opening the redirect may show the equivalent regular expression instead of the original wildcard text.

Programmi di installazione script modello
Questa pagina consente di specificare script di terze parti da installare quando viene creato un nuovo server virtuale utilizzando questo modello. Per ogni script, è necessario inserire le seguenti informazioni:

copione
Il nome e la versione di uno script da installare
Percorso URL
Un percorso nel sito Web del server virtuale per installare lo script, come /phpmyadmin . Ciò determinerà anche la directory in ~/public_html che contiene i file dello script.
Nome DB
Il nome del database che lo script utilizzerà per archiviare i dati. Le opzioni sono:
Nessuna
Questo dovrebbe essere selezionato per gli script che non richiedono un database, come Galleria .
Database principale
Il database dello script sarà il database primario del server virtuale del tipo scelto.
Altro
Se si desidera che questo script utilizzi il proprio database privato, è possibile selezionare questa opzione e immettere il modello per un database nel campo di testo adiacente. Ad esempio, è possibile immettere $ {DB} _phpnuke . Se il database risultante non esiste, verrà creato come parte del processo di installazione dello script.
Tipo di DB
Il tipo di database utilizzato per i dati degli script, se selezionato.

Vorlage PHP-Konfigurationsdatei
Wenn ein virtueller Server PHP-Skripte als Serverbesitzer ausführt, kopiert Virtualmin standardmäßig die globale PHP-Konfigurationsdatei des Systems (normalerweise /etc/php.ini ) in das Verzeichnis ~/etc der Domäne. Dadurch können PHP-Optionen pro Server unterschiedlich eingestellt werden.

Mit dieser Einstellung können Sie einen anderen Pfad für die zu kopierende PHP-Basiskonfigurationsdatei eingeben. Darüber hinaus können Sie in dieser Vorlagendatei Virtualmin-Ersetzungen wie ${DOM} verwenden, die beim Kopieren ersetzt werden. Andere nützliche Substitutionen sind:

${DOM} Der Domainname, z. B. foo.com
${USER} Das Administrations-Login des Servers, z. B. foo
${HOME} Das Home-Verzeichnis des Servers, wie /home/foo
${EMAILTO} Die Kontakt-E-Mail-Adresse des virtuellen Servers
Außerdem können bedingte Blöcke wie ${IF-MAIL} ... ${ENDIF-MAIL} und ${IF-WEB} ... ${ELSE-WEB} ... {$ENDIF-WEB} verwendet werden.

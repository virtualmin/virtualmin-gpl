Buckets de Amazon S3

Esta página enumera todos los buckets de Amazon S3 propiedad de las cuentas de S3 que Virtualmin está configurado para usar en la página Proveedores de almacenamiento en la nube. Los depósitos también se pueden crear y editar desde esta página, y se pueden editar las reglas del ciclo de vida de control de acceso y eliminación automática que se aplican a los archivos que contienen.

El proceso de copia de seguridad de Virtualmin creará automáticamente nuevos depósitos según sea necesario, pero en la ubicación de almacenamiento predeterminada de S3. La creación de un depósito desde esta página le permite seleccionar una ubicación de almacenamiento más cercana a su sistema Virtualmin.

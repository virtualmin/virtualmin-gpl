Префикс для дополнительных баз данных
По умолчанию Virtualmin позволит владельцам серверов создавать дополнительные базы данных с любыми именами. Однако это поле можно использовать для принудительного использования префикса. Хорошее значение - ${PREFIX} _, которое заставит имя базы данных начинаться с первой части имени домена.

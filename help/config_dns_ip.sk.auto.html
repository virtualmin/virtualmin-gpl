Predvolená adresa IP pre záznamy DNS
DNS záznamy sa automaticky generujú pre nových virtuálnych hostiteľov a za normálnych okolností budete chcieť, aby sa zhodovali s vyššie uvedenou IP virtuálneho servera. Ak je však váš server za smerovačom NAT, prevádzkujete proxy server, server na vyrovnávanie záťaže alebo nejaký iný typ zariadenia, ktoré vedie k tomu, že váš server funguje na inej IP adrese, než akú svet vidí pri komunikácii s vašimi doménami, tu možno bude potrebné zmeniť hodnotu. Ďalším prípadom, kedy to možno budete chcieť zmeniť, je, ak na vývojový server pridávate domény, ktoré budú mať pri uvedení do prevádzky odlišné adresy. Tu môžete vyplniť produkčnú alebo verejnú IP adresu, namiesto toho, aby Virtualmin používal lokálnu IP adresu.

Ak je vybratá možnosť Automaticky zistiť externú adresu , Virtualmin sa pokúsi zistiť IP v čase vytvorenia virtuálneho servera. www.virtualmin.com požiadavky HTTP na stránku www.virtualmin.com, ktorá odpovie s adresou, z ktorej vidí spojenie.

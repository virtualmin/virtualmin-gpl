Дней до истечения срока обновления сертификатов Let's Encrypt
По умолчанию Virtualmin обновляет сертификаты SSL Let's Encrypt до того, как срок их действия истечет. Эту опцию можно использовать для контроля того, как далеко до истечения срока будет предприниматься попытка продления, если по умолчанию установлено слишком рано.

Môže upravovať virtuálny server
Umožňuje úpravu funkcií virtuálneho servera, jeho hesla a popisu.
Môže spravovať používateľov
Umožňuje používateľom poštových schránok, FTP a databáz prezerať, upravovať, vytvárať a mazať.
Dokáže spravovať aliasy
Umožňuje prezerať, upravovať, vytvárať a odstraňovať e-mailové aliasy.
Dokáže spravovať databázy
Umožňuje pridať ďalšie databázy MySQL a PostgreSQL na tento virtuálny server alebo z neho odstrániť.
Môže inštalovať skripty
Umožňuje inštaláciu skriptov tretích strán na tento virtuálny server.
Môže zmeniť IP adresu
Ak má tento server súkromnú IP adresu, táto možnosť umožňuje jej zmenu.
Môže spravovať certifikáty SSL
Umožňuje odovzdanie alebo opätovné vygenerovanie certifikátu SSL a kľúča pre tento server.
Môže upravovať rámec preposielania
V prípade webových stránok určených len na preposielanie táto možnosť umožňuje úpravu cieľa a kódu HTML pre rámec preposielania.
Môže spravovať ďalších správcov
Umožňuje prezerať, vytvárať, upravovať a odstraňovať ďalšie správcovské účty vlastníkov servera.
Dokáže nakonfigurovať doručovanie spamu a vírusov
Umožňuje upravovať miesta doručenia spamu a vírusov a skóre.
Môžete si vybrať verzie PHP
Pre domény s PHP, ktoré bežia cez CGI alebo fastCGI, umožňuje nastavenie verzií PHP pre jednotlivé adresáre.
Môže upraviť nastavenia e-mailu
Umožňuje konfiguráciu automatického vytvárania BCC, režimu kopírovania aliasu a ďalších možností súvisiacich s e-mailom.
Môže robiť zálohy
Umožňuje vlastnené virtuálne servery zálohovať na vzdialený server FTP alebo SSH.
Môže plánovať zálohovanie
Umožňuje zálohovanie virtuálnych serverov podľa pravidelného plánu.
Dokáže obnoviť zálohy
Umožňuje obnoviť obsah domovského adresára a databázy vlastnených virtuálnych serverov.
Môže vybrať zdieľané adresy IP
Umožňuje výber globálne definovaných zdieľaných adries IP pri vytváraní virtuálnych serverov.
Môže vytvárať catchall aliasy
Umožňuje vytváranie e-mailových aliasov, ktoré preposielajú všetku poštu do domény na inú doménu alebo adresu.
Môže použiť editor webových stránok
Umožňuje úpravu webových stránok prostredníctvom používateľského rozhrania Virtualmin.
Môže upravovať vzdialených hostiteľov databázy
Umožňuje zadávanie vzdialených klientskych systémov, ktoré sa môžu pripojiť k MySQL databázam domény.
Môže zakázať virtuálne servery
Umožňuje dočasné vypnutie serverov.
Môže vymazať virtuálne servery
Umožňuje úplne vymazať vlastnené servery vrátane všetkých podserverov a aliasov.

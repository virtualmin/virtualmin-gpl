Ejecute scripts CGI usando
Cuando se instala el servidor fcgiwrap, esta opción le permite seleccionar si siempre se utilizará para ejecutar scripts CGI para nuevos dominios. De lo contrario, se utilizará suexec si está instalado y configurado correctamente.

Fcgiwrap es generalmente más rápido para ejecutar scripts CGI, pero requiere que cada servidor virtual ejecute su propio proceso de servidor que consume RAM adicional.

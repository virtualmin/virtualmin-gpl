Générer un certificat SSL pour les domaines sans SSL activé?
Par défaut, Virtualmin ne créera pas de certificat SSL pour les nouveaux serveurs virtuels sauf si la fonction SSL est activée - cependant, un certificat peut être ajouté plus tard. Lorsque cette option est activée, tous les nouveaux domaines recevront un certificat pouvant être utilisé par d'autres services comme Dovecot, Postfix ou des proxys SSL externes.

Prefixo para administradores extras
Este campo pode ser usado para forçar contas de administração extras a terem nomes de usuário iniciados com algum prefixo. Isso é útil para identificar a qual servidor um usuário Webmin específico está associado. Um valor comum para esse campo é ${USER} _, o que força os administradores extras a iniciarem com o mesmo nome que o login principal do servidor virtual.

Editar bancos de dados
Esta página lista todos os bancos de dados pertencentes ao servidor virtual e permite que você crie novos, até o limite permitido para o usuário conectado. Você também pode excluir um ou mais bancos de dados.

Para visualizar as tabelas e os dados de um banco de dados, clique no botão "Gerenciar". Se o phpMyAdmin estiver instalado neste servidor virtual, você também verá um botão "Gerenciar no phpMyAdmin". Se o phpMyAdmin estiver instalado em um subservidor, a instalação do phpMyAdmin do subservidor será usada em vez da instalação do servidor principal.

Para ver detalhes adicionais sobre um banco de dados, incluindo seu tamanho e número de tabelas, clique em seu nome.

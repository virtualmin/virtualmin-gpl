Editar bancos de dados
Esta página lista todos os bancos de dados pertencentes ao servidor virtual e permite que você crie novos, até o limite permitido para o usuário conectado. Você também pode excluir um ou mais bancos de dados.

Para visualizar as tabelas e os dados de um banco de dados, clique no botão "Gerenciar". Se o phpMyAdmin estiver instalado neste servidor virtual, você também verá um botão "Gerenciar no phpMyAdmin". Se o servidor virtual for um subservidor e tiver sua própria instalação do phpMyAdmin, essa será usada em vez da do servidor principal. Se não houver uma instalação por servidor, mas uma instalação padrão do phpMyAdmin tiver sido definida no sistema, essa instância compartilhada será usada.

Para ver detalhes adicionais sobre um banco de dados, incluindo seu tamanho e número de tabelas, clique em seu nome.

Редактировать базы данных
На этой странице отображается список всех баз данных, принадлежащих виртуальному серверу, и вы можете создавать новые в пределах лимита, установленного для вошедшего в систему пользователя. Вы также можете удалить одну или несколько баз данных.

Чтобы просмотреть таблицы и данные базы данных, нажмите кнопку «Управление». Если для этого виртуального сервера установлен phpMyAdmin, вы также увидите кнопку «Управление в phpMyAdmin». Если виртуальный сервер является подсервером и имеет собственную установку phpMyAdmin, будет использоваться именно она, а не установка родительского сервера. Если нет установки phpMyAdmin для каждого сервера отдельно, но установлен системный экземпляр phpMyAdmin по умолчанию, будет использоваться этот общий экземпляр.

Чтобы просмотреть дополнительные сведения о базе данных, включая ее размер и количество таблиц, щелкните по ее названию.

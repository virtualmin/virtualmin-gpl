Envoyer une réponse automatique
Lorsque cette option est cochée, tout e-mail adressé à cet utilisateur déclenchera une réponse automatique contenant le texte saisi dans la zone de texte adjacente. Le texte peut inclure des en-têtes d'e-mail supplémentaires en haut, comme:
Subject: Votre sujet de réponse automatique
Cc: foo@bar.com

Ceci est votre message de réponse automatique.
Bonne journée!
Pour envoyer une réponse HTML automatique, commencez le message avec une <html> .

Le contenu du message d'origine peut être inclus dans la réponse à l'aide de variables telles que $SUBJECT , $TO et $FROM .

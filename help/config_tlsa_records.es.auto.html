¿Crear registros DNS TLSA para certificados SSL?
Cuando está habilitado, Virtualmin agregará un registro DNS TLSA para todos los nombres de host que usan certificados SSL para servidores virtuales recién creados. Esto permite a los clientes validar que el certificado SSL presentado en su servidor web es el mismo que el alojado en su sistema, lo que evita ciertos ataques de intermediario.

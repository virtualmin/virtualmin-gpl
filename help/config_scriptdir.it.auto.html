Modello per la directory di script predefinita
Per impostazione predefinita, Virtualmin installerà i nuovi script in una sottodirectory con lo stesso nome del codice dello script, come phpadmin . Questo campo consente di inserire un modello per una directory predefinita diversa, come scripts/${SCRIPTNAME} . È possibile utilizzare anche tutte le solite variabili modello come ${DOM} e ${USER}.

Пользовательская ссылка
Каждая настраиваемая ссылка имеет следующие атрибуты, которые можно настроить в этой форме:

Заголовок ссылки
Текст ссылки. Это может включать замены, такие как $DOM и $USER, которые заменяются данными выбранного домена.
Целевой URL
Полный URL, на который указывает ссылка. Опять же, можно использовать такие замены, как $DOM.
Открыть в окне
Если выбрано Same , ссылка откроется в правой части окна. Если выбрано Новое , оно откроется в отдельном окне браузера верхнего уровня.
категория
Если вы определили какие-либо пользовательские категории для ссылок, вы можете выбрать категорию для каждой ссылки в этом столбце.
Домены и пользователи, на которые отображаются ссылки, также могут контролироваться с помощью следующих полей на этой странице:

Типы пользователей Virtualmin
Чтобы ограничить ссылки на главного администратора, владельцев доменов или посредников Virtualmin, установите соответствующие флажки в этом поле.
Виртуальные серверы из шаблона
Только когда в левом меню будет выбран виртуальный сервер, созданный с использованием выбранного шаблона, эта ссылка будет видимой.
Виртуальные серверы с функцией
Ссылка появится только при выборе виртуального сервера с выбранной функцией.

Clave pública SSH
Este campo se puede utilizar para otorgar acceso SSH autenticado mediante una clave pública al usuario Unix del nuevo servidor virtual. Las opciones son:

Sin clave predeterminada
No configure una clave SSH. Solo se permitirán inicios de sesión con contraseña.

Genera claves públicas y privadas
Cree un nuevo par de claves SSH para este servidor virtual y permita los inicios de sesión con esa clave. El propietario del servidor debe copiar la clave privada después de la creación.

Utilice la clave pública a continuación
Se permitirán los inicios de sesión SSH utilizando la clave privada que corresponde a la clave pública ingresada en el cuadro de texto a continuación.

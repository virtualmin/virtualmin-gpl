Predvolený režim vykonávania PHP
Táto možnosť určuje, či skripty PHP na nových virtuálnych serveroch spúšťa používateľ webového servera Apache (zvyčajne httpd alebo apache ) alebo používateľ systému Unix, ktorý vlastní virtuálny server. Prvý z nich je rýchlejší, pretože to znamená, že na spustenie kódu PHP nie je potrebné spúšťať žiadne ďalšie procesy, ale je menej bezpečný, pretože skripty čítajú a zapisujú súbory ako iný používateľ ako vlastník domény.

Ak to váš systém podporuje, voľba FCGId je najlepšia, pretože spúšťa PHP skripty rýchlo a ako vlastník domény. V opačnom prípade, pokiaľ váš systém nie je silne zaťažený alebo nie je problémom bezpečnosť medzi používateľmi, odporúča sa možnosť CGI wrapper.

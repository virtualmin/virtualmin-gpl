Voer CGI-scripts uit met
Wanneer de fcgiwrap- server is geïnstalleerd, kunt u met deze optie selecteren of deze altijd zal worden gebruikt om CGI-scripts voor nieuwe domeinen uit te voeren. Anders wordt suexec gebruikt als het correct is geïnstalleerd en geconfigureerd.

FCGIwrap is over het algemeen sneller om CGI-scripts uit te voeren, maar vereist dat elke virtuele server zijn eigen serverproces uitvoert dat extra RAM verbruikt.

Archivo de plantilla para correo electrónico a nuevos buzones
Si los nuevos buzones creados se deben enviar automáticamente un mensaje, puede especificarlo aquí. Puede ser el predeterminado, que puede editarse utilizando la página Nuevo correo electrónico de buzón de Virtualmin, o puede proporcionar otro archivo de texto en cualquier parte del sistema. Sustitución de variables usando las siguientes variables: ${MAILBOX}, ${USER}, ${DOM}, ${FTP} y ${HOME}. También puede usar los operadores de bloques lógicos especiales $IF-MAIL ... $ENDIF-MAIL y $IF-WEB ... $ELSE-WEB ... {$ENDIF-WEB} para permitir que su mensaje cree un mensaje apropiado según las características a las que el usuario tiene acceso. Si está utilizando DAV, el bloque ${IF-VIRTUALMIN-DAV} .. ${ENDIF-VIRTUALMIN-DAV} se puede usar para que aparezca texto diferente dependiendo de si el usuario tiene DAV activo o no.

File modello per e-mail a nuove cassette postali
Se è necessario inviare automaticamente un messaggio alle nuove cassette postali create, è possibile specificarlo qui. Può essere l'impostazione predefinita, che può essere modificata utilizzando la pagina E-mail nuova cassetta postale di Virtualmin oppure è possibile fornire un altro file di testo in qualsiasi punto del sistema. Sostituzione delle variabili usando le seguenti variabili: ${MAILBOX}, ${USER}, ${DOM}, ${FTP} e ${HOME}. È inoltre possibile utilizzare gli operatori del blocco logico speciale $IF-MAIL ... $ENDIF-MAIL e $IF-WEB ... $ELSE-WEB ... {$ENDIF-WEB} per consentire al messaggio di creare un messaggio appropriato in base alle funzionalità a cui l'utente ha accesso. Se stai usando DAV, il blocco ${IF-VIRTUALMIN-DAV} .. ${ENDIF-VIRTUALMIN-DAV} può essere usato per far apparire un testo diverso a seconda che l'utente abbia DAV attivo o meno.

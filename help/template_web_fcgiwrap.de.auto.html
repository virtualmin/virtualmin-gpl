Führen Sie CGI-Skripte mit aus
Wenn der fcgiwrap- Server installiert ist, können Sie mit dieser Option auswählen, ob er immer zum Ausführen von CGI-Skripten für neue Domänen verwendet wird. Andernfalls wird suexec verwendet, wenn es richtig installiert und konfiguriert ist.

FCGIwrap ist im Allgemeinen schneller beim Ausführen von CGI-Skripten, erfordert jedoch, dass jeder virtuelle Server seinen eigenen Serverprozess ausführt, der zusätzlichen RAM verbraucht.

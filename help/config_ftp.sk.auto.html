Je povolený virtuálny server ProFTPd založený na IP?
Povolí alebo zakáže nového virtuálneho hostiteľa FTP. Upozorňujeme, že FTP vyžaduje novú IP adresu pre anonymný hosting virtuálnej domény. Prevažná väčšina vlastníkov webových stránok túto funkciu nepotrebuje a nie je potrebná pre overený FTP prístup k súborom vlastníkov domén. Ak však niektoré domény budú vyžadovať virtuálny hosťovaný anonymný FTP prístup (tj ak prevádzkujú súborový server FTP pre softvér alebo sťahovanie súborov pre neprihlásených používateľov), možno budete chcieť nastaviť možnosť „Áno, ale nie predvolene“ .

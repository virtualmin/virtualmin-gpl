Шаблон PHP конфигурационного файла
По умолчанию, когда виртуальный сервер выполняет сценарии PHP как владелец сервера, Virtualmin копирует глобальный системный конфигурационный файл PHP системы (обычно /etc/php.ini ) в каталог ~/etc домена. Это позволяет по-разному устанавливать параметры PHP для каждого сервера.

Этот параметр можно использовать для ввода другого пути для базового файла конфигурации PHP для копирования. Кроме того, вы можете использовать замены Virtualmin, такие как $ {DOM} в этом файле шаблона, который будет заменен при копировании. Другие полезные замены включают в себя:

${DOM} Доменное имя, например, foo.com
${USER} Логин администратора сервера, например, foo
${HOME} Домашний каталог сервера, например /home/foo
${EMAILTO} Контактный адрес электронной почты виртуального сервера
Кроме того, могут использоваться условные блоки, такие как $ {IF-MAIL} ... $ {ENDIF-MAIL} и $ {IF-WEB} ... $ {ELSE-WEB} ... {$ ENDIF-WEB} .

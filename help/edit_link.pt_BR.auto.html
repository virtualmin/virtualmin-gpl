Link personalizado
Cada link personalizado possui os seguintes atributos que você pode configurar neste formulário:

Título do link
O texto do link. Isso pode incluir substituições como $DOM e $USER, que são substituídas por detalhes do domínio selecionado.
URL de destino
O URL completo para o qual o link aponta. Novamente, substituições como $DOM podem ser usadas.
Abrir na janela
Se Mesmo estiver selecionado, o link será aberto no quadro do lado direito. Se Novo for escolhido, ele será aberto em uma janela separada do navegador de nível superior.
Categoria
Se você definiu categorias personalizadas para links, poderá selecionar uma categoria para cada link nesta coluna.
Os domínios e usuários aos quais os links parecem também podem ser controlados usando os seguintes campos nesta página:

Tipos de usuário do Virtualmin
Para limitar links para o administrador principal, proprietários de domínio ou revendedores do Virtualmin, marque as caixas apropriadas neste campo.
Servidores virtuais do modelo
Somente quando um servidor virtual criado usando o modelo selecionado for escolhido no menu esquerdo, esse link estará visível.
Servidores virtuais com recurso
Somente quando um servidor virtual com o recurso selecionado for escolhido, o link será exibido.

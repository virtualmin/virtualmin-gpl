电子邮件到新邮箱的模板文件
如果应该自动向创建的新邮箱发送消息，则可以在此处指定。它可以是默认值，可以使用Virtualmin的“ 新建邮箱电子邮件”页面进行编辑，或者可以在系统上的任何位置提供另一个文本文件。使用以下变量进行变量替换：${MAILBOX}，${USER}，${DOM}，${FTP}和${HOME}。您还可以使用特殊逻辑块运算符$IF-MAIL ... $ENDIF-MAIL和$IF-WEB ... $ELSE-WEB ... {$ENDIF-WEB}允许您的消息创建适当的消息根据用户有权使用的功能。如果您使用的是DAV，则可以使用${IF-VIRTUALMIN-DAV} .. ${ENDIF-VIRTUALMIN-DAV}块来显示不同的文本，具体取决于用户是否激活了DAV。

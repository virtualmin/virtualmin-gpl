Zde lze použít následující substituce. Při použití šablony budou nahrazeny skutečnými hodnotami z domény:

${DOM} Název domény, například foo.com
${IP} IP adresa virtuálního serveru, jako je 1.2.3.4
${DNS_IP} Externí adresa IP virtuálního serveru použitá v záznamech DNS
${USER} Přihlašovací údaje pro správu serveru, například foo
${GROUP} Skupina Unix pro tento virtuální server.
${UID} and ${GID} ID uživatele serveru Unix a ID skupiny
${HOME} Domovský adresář serveru, například /home/foo
${EMAILTO} Kontaktní e-mailová adresa virtuálního serveru
${PARENT} Pokud se jedná o sub-server, jedinečné ID nadřazené domény.
${ALIAS} Pokud se jedná o alias, jedinečné ID cílové domény.
${ID} Jedinečné číselné ID pro tuto doménu.
${CREATED} Časové razítko, ve kterém byla tato doména vytvořena.
${DB} Název databáze MySQL nebo PostgreSQL.
Kromě toho lze použít podmíněné bloky jako ${IF-MAIL} ... ${ENDIF-MAIL} a ${IF-WEB} ... ${ELSE-WEB} ... {$ENDIF-WEB} .

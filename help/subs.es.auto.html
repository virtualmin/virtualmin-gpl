Las siguientes sustituciones se pueden utilizar aquí. Se reemplazarán con los valores reales del dominio cuando se use la plantilla:

${DOM} El nombre de dominio, como foo.com
${IP} La dirección IP del servidor virtual, como 1.2.3.4
${DNS_IP} La dirección IP externa del servidor virtual, utilizada en registros DNS
${USER} El inicio de sesión de administración del servidor, como foo
${GROUP} El grupo Unix para este servidor virtual.
${UID} and ${GID} El ID de usuario Unix del servidor y el ID de grupo respectivamente
${HOME} El directorio de inicio del servidor, como /home/foo
${EMAILTO} La dirección de correo electrónico de contacto del servidor virtual
${PARENT} Si se trata de un servidor secundario, la ID única del dominio principal.
${ALIAS} Si se trata de un alias, la ID única del dominio de destino.
${ID} La identificación numérica única para este dominio.
${CREATED} La marca de tiempo en la que se creó este dominio.
${DB} Nombre de la base de datos MySQL o PostgreSQL.
Además, se pueden usar bloques condicionales como ${IF-MAIL} ... ${ENDIF-MAIL} y ${IF-WEB} ... ${ELSE-WEB} ... {$ENDIF-WEB} .

Auch verschlüsselte Backups signieren?
Wenn ein Backup-Verschlüsselungsschlüssel ausgewählt ist, steuert dieses Kontrollkästchen, ob dieser zusätzlich digital signiert wird. Das Erstellen einer Signatur verlängert die Backup-Dauer und beansprucht mehr Rechenzeit, gewährleistet aber, dass das Backup nicht manipuliert wurde, falls es auf einem externen Speicher abgelegt wird, über den Sie keine Kontrolle haben. Zudem können nur signierte Backups von Domaininhabern wiederhergestellt werden.

HTML del sitio web deshabilitado
Este campo se puede usar para personalizar el mensaje que aparece cuando se conecta a un sitio web para un servidor virtual deshabilitado. El mensaje predeterminado simplemente indica Sitio web deshabilitado . Puede usar sustituciones típicas como ${DOM} y ${USER} para obtener detalles del dominio que se deshabilita, ${DISABLED_WHY} para obtener la razón por la que se deshabilitó el dominio y ${DISABLED_TIME} para obtener el momento en que la deshabilitación fue hecho

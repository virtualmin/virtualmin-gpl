Nom d'hôte pour l'enregistrement MX
Par défaut, l'enregistrement DNS MX (qui contrôle le lieu de livraison des e-mails) pour les nouveaux domaines sera mail. $ DOMAIN, auquel sera attribuée la même adresse IP que le site Web du serveur virtuel. Cependant, vous pouvez utiliser ce champ pour spécifier un nom d'hôte de serveur de messagerie différent - il est généralement utilisé pour avoir un enregistrement MX cohérent dans tous les domaines. Assurez-vous de saisir un nom d'hôte qui résout votre système Virtualmin, sinon le courrier électronique ne sera pas remis.

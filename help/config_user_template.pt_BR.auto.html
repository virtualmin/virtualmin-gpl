Arquivo de modelo para email para novas caixas de correio
Se as novas caixas de correio criadas receberem automaticamente uma mensagem, você pode especificá-la aqui. Pode ser o padrão, que pode ser editado usando a página Novo email da caixa de correio do Virtualmin, ou você pode fornecer outro arquivo de texto em qualquer lugar do sistema. Substituição de variável usando as seguintes variáveis: ${MAILBOX}, ${USER}, ${DOM}, ${FTP} e ${HOME}. Você também pode usar os operadores especiais de blocos lógicos $IF-MAIL ... $ENDIF-MAIL e $IF-WEB ... $ELSE-WEB ... {$ENDIF-WEB} para permitir que sua mensagem crie uma mensagem apropriada com base nos recursos aos quais o usuário tem acesso. Se você estiver usando DAV, o bloco ${IF-VIRTUALMIN-DAV} .. ${ENDIF-VIRTUALMIN-DAV} poderá ser usado para fazer com que textos diferentes apareçam, dependendo se o usuário tem ou não DAV ativo.

Файл шаблона для электронной почты в новые почтовые ящики
Если созданным новым почтовым ящикам автоматически отправляется сообщение, вы можете указать его здесь. Это может быть значение по умолчанию, которое может быть отредактировано с помощью страницы электронной почты New Mailbox в Virtualmin, или вы можете предоставить другой текстовый файл в любом месте системы. Подстановка переменных с использованием следующих переменных: ${MAILBOX}, ${USER}, ${DOM}, ${FTP} и ${HOME}. Вы также можете использовать специальные операторы логических блоков $IF-MAIL ... $ENDIF-MAIL и $IF-WEB ... $ELSE-WEB ... {$ENDIF-WEB}, чтобы ваше сообщение могло создать соответствующее сообщение. на основе функций, к которым пользователь имеет доступ. Если вы используете DAV, блок ${IF-VIRTUALMIN-DAV} .. ${ENDIF-VIRTUALMIN-DAV} может использоваться для отображения другого текста в зависимости от того, активен ли пользователь DAV.

新しいメールボックスへのメールのテンプレートファイル
作成した新しいメールボックスに自動的にメッセージを送信する必要がある場合は、ここで指定できます。これはデフォルトにすることができ、Virtualminの[ 新しいメールボックスの電子メール]ページを使用して編集できます。または、システム上の任意の場所に別のテキストファイルを提供できます。次の変数を使用した変数置換：${MAILBOX}、${USER}、${DOM}、${FTP}、および${HOME}。特殊な論理ブロック演算子$IF-MAIL ... $ENDIF-MAILおよび$IF-WEB ... $ELSE-WEB ... {$ENDIF-WEB}を使用して、メッセージで適切なメッセージを作成することもできます。ユーザーがアクセスできる機能に基づいています。 DAVを使用している場合、${IF-VIRTUALMIN-DAV} .. ${ENDIF-VIRTUALMIN-DAV}ブロックを使用して、ユーザーがDAVをアクティブにしているかどうかに応じて異なるテキストを表示できます。

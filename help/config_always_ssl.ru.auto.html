Создать сертификат SSL для доменов без включенного SSL?
По умолчанию Virtualmin не будет создавать сертификат SSL для новых виртуальных серверов, если на них не включена функция SSL, однако сертификат можно добавить позже. Когда эта опция включена, все новые домены получат сертификат, который может использоваться другими службами, такими как Dovecot, Postfix или внешними SSL-прокси.

Benutzerdefinierten Link
Jeder benutzerdefinierte Link verfügt über die folgenden Attribute, die Sie in diesem Formular konfigurieren können:

Link-Titel
Der Text des Links. Dies kann Ersetzungen wie $DOM und $USER umfassen, die durch Details der ausgewählten Domain ersetzt werden.
Ziel-URL
Die vollständige URL, auf die der Link verweist. Auch hier können Substitutionen wie $DOM verwendet werden.
Im Fenster öffnen
Wenn Gleich ausgewählt ist, wird der Link im rechten Rahmen geöffnet. Wenn Neu ausgewählt ist, wird es in einem separaten Browserfenster der obersten Ebene geöffnet.
Kategorie
Wenn Sie benutzerdefinierte Kategorien für Links definiert haben, können Sie für jeden Link in dieser Spalte eine Kategorie auswählen.
Die Domänen und Benutzer, zu denen Links angezeigt werden, können auch über die folgenden Felder auf dieser Seite gesteuert werden:

Virtualmin-Benutzertypen
Aktivieren Sie die entsprechenden Kontrollkästchen in diesem Feld, um die Links zum Hauptadministrator, zu Domänenbesitzern oder zu Virtualmin-Wiederverkäufern einzuschränken.
Virtuelle Server aus Vorlage
Nur wenn ein virtueller Server, der mit der ausgewählten Vorlage erstellt wurde, aus dem linken Menü ausgewählt wird, wird dieser Link angezeigt.
Virtuelle Server mit Funktion
Nur wenn ein virtueller Server mit der ausgewählten Funktion ausgewählt ist, wird der Link angezeigt.

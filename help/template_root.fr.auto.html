Répertoire racine dans le sélecteur de fichiers
Cette option spécifie le répertoire que le sélecteur de fichiers affichera lors de sa première ouverture. Avec les autorisations appropriées, l'utilisateur peut ensuite naviguer vers d'autres répertoires. La substitution de variable peut être effectuée à l'aide de la variable ${HOME} pour spécifier le répertoire personnel du nouveau domaine.

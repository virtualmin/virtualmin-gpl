FTP login aangezet?
Wanneer deze optie is ingesteld op Ja krijgt de gebruiker de shell, waarmee hij kan inloggen via FTP. Wanneer dit is ingesteld op Nee, word zijn shell ingesteld op en zullen FTP inlog pogingen worden geweigerd.

Hiermee word aangenomen dat de FTP shell zich bevind in de /etc/shells file, of op een andere wijze word toegestaan door de FTP server die aan het werk is op uw systeem.

Может настроить прокси?
По умолчанию, если для этого параметра установлено значение «Да», торговый посредник может создать виртуальный сервер, который проксирует все запросы на другой URL-адрес, и настроить проксирование для выбранных путей в существующих доменах. Выбор «Нет» предотвратит использование этих функций.

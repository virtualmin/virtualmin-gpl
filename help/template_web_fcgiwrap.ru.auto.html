Запускайте сценарии CGI, используя
Когда установлен сервер fcgiwrap, этот параметр позволяет выбрать, будет ли он всегда использоваться для запуска сценариев CGI для новых доменов. В противном случае будет использоваться suexec , если он установлен и настроен правильно.

Fcgiwrap обычно быстрее запускает сценарии CGI, но требует, чтобы каждый виртуальный сервер запускал свой собственный серверный процесс, который потребляет дополнительную оперативную память.

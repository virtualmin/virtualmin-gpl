ドメインに変更を加えた後に実行するコマンド
ドメインを変更した後に何らかのアクション（作成、変更、または削除）を実行したい場合は、ここで実行するコマンドを指定できます。複数のアクションを実行する場合、実行するコマンドはスクリプトにすることができます。このコマンドは、ドメインに固有の特別な環境変数（VIRTUALSERVER_DOM、VIRTUALSERVER_IP、VIRTUALSERVER_USER、VIRTUALSERVER_HOME、VIRTUALSERVER_PASSなど）にアクセスできます。 / etc / webmin / virtual-server / domains / $DOMファイルにあるすべての変数は、環境変数として使用できます。

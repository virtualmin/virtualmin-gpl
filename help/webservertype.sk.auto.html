Typ webového servera
Virtuálny webový server môže byť založený na názve alebo na IP. V drvivej väčšine prípadov postačuje virtuálny server založený na názve. Virtuálny server založený na IP je potrebné použiť iba vtedy, keď má byť hosťovaná stránka SSL alebo musí byť pre doménu poskytnutý anonymný FTP adresár.

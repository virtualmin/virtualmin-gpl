Konfigurieren Sie Apache für die Verwendung
Diese Option legt fest, ob Apache so konfiguriert werden soll, dass eine einzelne Datei verwendet wird, die das SSL-Zertifikat für einen virtuellen Server und das CA-Zertifikat kombiniert, oder ob sie in separate Dateien aufgeteilt werden sollen. Standardmäßig werden sie nur kombiniert, wenn Apache 2.4.8 oder höher ausgeführt wird.

Имя базы данных по умолчанию
Virtualmin позволяет очень гибкое создание базы данных и разрешений. В этом поле можно указать имя по умолчанию для баз данных, созданных как часть доменов, созданных из этого шаблона. По умолчанию Virtualmin использует переменную ${PREFIX}, которая является первой частью имени домена и, как правило, также будет именем входа владельца домена для Webmin. Может быть полезно изменить это, в некоторых случаях включить префикс, который указывает шаблон, из которого он был сгенерирован, или включить дополнительную информацию о базе данных. Можно комбинировать переменные и любые символы, которые разрешены в базе данных PostgreSQL и / или MySQL (в зависимости от того, какие базы данных доступны).

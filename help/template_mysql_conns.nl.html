Maximum verbindingen voor domein eigenaars
Dit veld bepaald het aantal samengaande verbindingen naar MySQL die een virtuele server eigenaar account kan maken. Dit is inclusief de verbindingen van scripts die geinstalleerd zijn door Virtualmin, zoals Wordpress of Drupal. De standaard is om een ongelimiteerd aantal verbindingen toe te staan, maar omdat elke verbinding binnen de MySQL server proces geheugen gebruikt is het verstandig op een druk systeem dit aantal te limiteren.

De limiet die hier word ingesteld word alleen toegevoegd wanneer er een nieuwe virtuele server word aangemaakt, of wanneer een MySQL login word aangezet voor een bestaande server.

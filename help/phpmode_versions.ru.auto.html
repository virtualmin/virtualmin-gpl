Версии PHP
Эту таблицу можно использовать для настройки версии PHP по умолчанию для этого виртуального сервера и, при необходимости, версий, используемых для выбранных подкаталогов. Эта возможность может быть полезна, если вам нужно запустить несколько скриптов, поддерживающих только определенные версии PHP.

Nastaviť virtuálny FTP založený na IP?
Keď je táto funkcia povolená, pre tento virtuálny server Virtualmin sa vytvorí virtuálny server ProFTPd. Toto možno použiť na nastavenie anonymnej lokality FTP, ktorá je súkromná pre tento server. Vyžaduje si to však, aby mal server svoju IP adresu.

Táto funkcia je potrebná len vtedy, ak chcete poskytnúť prístup na základe názvu k anonymnému serveru FTP. Takmer nikto nepotrebuje túto funkciu povoliť (to pravdepodobne znamená vy). Túto funkciu nemusíte aktivovať, ak chcete iba poskytnúť FTP prístup pre účet vlastníka domény, pretože každý používateľ s shellovým účtom bude mať automaticky FTP prístup do svojho domovského adresára.

Inými slovami: Toto takmer určite nepotrebujete, aj keď si myslíte, že možno áno.

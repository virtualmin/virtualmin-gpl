Malfil for e-post til oppdaterte postbokser
Hvis postkasser automatisk skal sendes en melding når de blir oppdatert, kan du angi den her. Det kan være standard, som kan redigeres ved å bruke den oppdaterte e-postadressen for e-postkassen til Virtualmin, eller du kan gi en annen tekstfil hvor som helst på systemet. Variabel substitusjon med følgende variabler: ${MAILBOX}, ${USER}, ${DOM}, ${FTP} og ${HOME}. Du kan også bruke de spesielle logikkblokkoperatørene $IF-MAIL ... $ENDIF-MAIL og $IF-WEB ... $ELSE-WEB ... {$ENDIF-WEB} for å la meldingen din lage en passende melding basert på funksjonene brukeren har tilgang til.

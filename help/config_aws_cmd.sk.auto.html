Úplná cesta k príkazu aws pre operácie S3
Ak aws a toto pole je nastavené na úplnú cestu k nemu (napríklad /usr/bin/aws ), Virtualmin ho zavolá, aby vykonal operácie S3 pre zálohy, ako sú nahrané a stiahnuté súbory. Pretože tento príkaz bol vyvinutý spoločnosťou Amazon, je spoľahlivejší ako S3 API zabudované do Virtualminu.

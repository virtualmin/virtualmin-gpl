Configurer Apache à utiliser
Cette option détermine si Apache doit être configuré pour utiliser un seul fichier qui combine le certificat SSL pour un serveur virtuel et le certificat CA, ou s'ils doivent être divisés en fichiers séparés. La valeur par défaut est de les combiner uniquement lors de l'exécution d'Apache 2.4.8 ou version ultérieure.

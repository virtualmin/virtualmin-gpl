Dávkové vytváranie serverov
Táto stránka vám umožňuje vytvoriť viacero virtuálnych serverov naraz z textového súboru. Tento súbor musí obsahovať jeden server na riadok a každý riadok musí mať nasledujúci formát:

domain : description : password :[ username ]:[ parent-domain ]:[ ip-address ]:[ alias-domain ]

Význam polí je:

doména
Názov domény pre virtuálny server, ktorý sa má vytvoriť, napríklad foo.com .
popis
Ľudsky čitateľný popis, ako napríklad Joe's Website .
heslo
Heslo pre prihlásenie do administrácie servera. Toto je potrebné iba pre nadradené servery.
užívateľské meno
Administračné používateľské meno pre server. Ak toto ponecháte prázdne, používateľské meno sa vyberie automaticky.
rodičovská doména
Názov domény nadradeného servera, pod ktorým sa má tento server vytvoriť. Ak toto pole ponecháte prázdne, nový server sa vytvorí ako samotný rodič.
IP adresa
Pre virtuálne servery, ktoré nepotrebujú vlastnú súkromnú IP, môže byť toto pole ponechané prázdne. V opačnom prípade musíte zadať platnú nepoužitú adresu alebo slovo allocate aby ju za vás vybral Virtualmin (ak je nakonfigurovaný).
alias-doména
Názov domény virtuálneho servera, pre ktorý by sa to malo stať aliasom. Ak sa vynechá, nový server bude buď najvyššej úrovne, alebo podserver.

Kvóty, šírka pásma a ďalšie limity pre vytvorené servery sú nastavené na základe šablóny, ktorú vyberiete v tomto formulári. Funkcie pre vytvorené servery sú určené zaškrtávacími políčkami funkcií, rovnako ako na normálnej stránke vytvorenia jedného servera.

Ak váš systém používa účty predajcov, môžete si vybrať, aby boli všetky virtuálne servery vytvorené vo vlastníctve predajcu. Táto možnosť je dostupná iba hlavnému správcovi.

Настройте Apache для использования
Этот параметр определяет, следует ли настроить Apache для использования одного файла, который объединяет сертификат SSL для виртуального сервера и сертификат CA, или их следует разделить на отдельные файлы. По умолчанию они объединяются только при запуске Apache 2.4.8 или новее.

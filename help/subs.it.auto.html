Le seguenti sostituzioni possono essere utilizzate qui. Verranno sostituiti con i valori effettivi del dominio quando viene utilizzato il modello:

${DOM} Il nome di dominio, ad esempio foo.com
${IP} L'indirizzo IP del server virtuale, come 1.2.3.4
${DNS_IP} L'indirizzo IP esterno del server virtuale, utilizzato nei record DNS
${USER} Il login di amministrazione del server, come foo
${GROUP} Il gruppo Unix per questo server virtuale.
${UID} and ${GID} L'ID utente Unix del server e l'ID gruppo rispettivamente
${HOME} La home directory del server, come /home/foo
${EMAILTO} L'indirizzo e-mail di contatto del server virtuale
${PARENT} Se si tratta di un server secondario, l'ID univoco del dominio padre.
${ALIAS} Se questo è un alias, l'ID univoco del dominio di destinazione.
${ID} L'ID numerico univoco per questo dominio.
${CREATED} Il timestamp in cui è stato creato questo dominio.
${DB} Nome del database MySQL o PostgreSQL.
Inoltre, è possibile utilizzare blocchi condizionali come ${IF-MAIL} ... ${ENDIF-MAIL} e ${IF-WEB} ... ${ELSE-WEB} ... {$ENDIF-WEB} .

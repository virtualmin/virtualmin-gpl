¿Almacenar contraseñas de texto sin cifrar?
Cuando esta opción se establece en Sí , Virtualmin almacenará las contraseñas de texto sin cifrar para los propietarios de dominios y buzones de correo. Esto permite que se muestren en la interfaz de usuario, pero agrega un riesgo de seguridad potencial si su sistema se ve comprometido. Si selecciona No , todos los servidores virtuales nuevos creados a partir de esta plantilla almacenarán solo contraseñas hash unidireccionales.

Dado que las aplicaciones web necesitan almacenar las contraseñas de la base de datos en texto sin formato, la contraseña de la base de datos para un usuario se establecerá en una nueva contraseña aleatoria si Virtualmin está configurado para no almacenar contraseñas en texto sin cifrar. Puede ver esta contraseña yendo a Editar bases de datos> Contraseñas .

As seguintes substituições podem ser usadas aqui. Eles serão substituídos pelos valores reais do domínio quando o modelo for usado:

${DOM} O nome do domínio, como foo.com
${IP} O endereço IP do servidor virtual, como 1.2.3.4
${DNS_IP} O endereço IP externo do servidor virtual, usado nos registros DNS
${USER} O login de administração do servidor, como foo
${GROUP} O grupo Unix para este servidor virtual.
${UID} and ${GID} O ID do usuário Unix e o ID do grupo do servidor, respectivamente
${HOME} O diretório inicial do servidor, como /home/foo
${EMAILTO} O endereço de email de contato do servidor virtual
${PARENT} Se este for um sub-servidor, o ID exclusivo do domínio pai.
${ALIAS} Se este for um alias, o ID exclusivo do domínio de destino.
${ID} O ID numérico exclusivo para este domínio.
${CREATED} O registro de data e hora em que este domínio foi criado.
${DB} Nome do banco de dados MySQL ou PostgreSQL.
Além disso, blocos condicionais como ${IF-MAIL} ... ${ENDIF-MAIL} e ${IF-WEB} ... ${ELSE-WEB} ... {$ENDIF-WEB} podem ser usados.
